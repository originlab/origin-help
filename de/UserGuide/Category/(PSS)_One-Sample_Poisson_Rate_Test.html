(PSS) Ereignisrate in Poisson-Modellen bei einer Stichprobe

PSS-OneSample-Poisson-Rate

Die Trennschärfe und der Stichprobenumfang für eine Ereignisrate in Poisson-Modellen bei einer Stichprobe widmen sich der Frage, wie viele Beobachtungen erforderlich sind, um eine festgelegte Differenz zwischen einer hypothetischen Rate und der wahren Rate bei einem gegebenen Konfidenzniveau zu erkennen. Von den Daten wird angenommen, dass sie einer Poisson-Verteilung folgen.

Allgemein gesagt können die Werkzeuge für Trennschärfe und Stichprobenumfang auf zwei Arten verwendet werden ­ zur Stichprobenschätzung und zur Berechnung der Trennschärfe. Ersteres kann verwendet werden, um den Umfang einer Stichprobe zu bestimmen und einen bestimmte Stufe für die Trennschärfe beim Aufbau eines Versuchs sicher zu stellen. Letzteres kann verwendet werden, um die Genauigkeit des Ergebnisses zu schätzen, nachdem ein Versuch für eine festgelegte Anzahl von Stichproben durchgeführt wurde.


Zum Bestimmen von Trennschärfe und Stichprobenumfang für eine Ereignisrate in Poisson-Modellen bei einer Stichprobe:

  1. Wählen Sie Statistik: Trennschärfe und Stichprobenumfang: (PSS) Ereignisrate in Poisson-Modellen bei einer Stichprobe. Der Dialog PSS_poissonTest1 wird geöffnet.
  2. Wählen Sie aus der Liste Berechnen die Option Trennschärfe oder Stichprobenumfang und dann Ihre restlichen Optionen.
  3. Nachdem Sie auf OK geklickt haben, wird ein Analysebericht erstellt.
