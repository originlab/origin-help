Minitools

Gadgets

Origin enthält mehrere Minitools für die Analyse von Daten, die in einem Diagramm gezeichnet sind. Auf die Minitools können Sie über das Menü Minitools bei aktivem Diagramm zugreifen.

Wenn Sie ein Minitool aus dem Menü auswählen, zeigt Origin auf dem Diagramm ein Rechteck, die so genannte grafische Datenauswahl Region of Interest (ROI), mit dem Sie grafisch die Daten auswählen, die Sie in Ihrer Analyse analysieren möchten. Entweder durch Doppelklick auf das ROI-Feld oder Im Ausklappmenü der grafischen Datenauswahl (ROI) ist bei den meisten Minitools ein Dialog für die Einstellungen verfügbar. Sie können Ihre Einstellungen auch in einer Designdatei speichern, wenn Sie sie später erneut verwenden möchten.

In einem Diagramm werden mehrere Minitools unterstützt. Sie können die Farbe jeder ROI benutzerdefiniert anpassen, um sie voneinander unterscheiden zu können. . Klicken Sie auf eine bestimmte grafische Datenauswahl, um sie in den Vordergrund zu bringen. Wenn eine ROI ausgewählt ist, können Sie auch die Tab-Taste verwenden, um von einer grafischen Datenauswahl zur anderen zu wechseln.

Hinweis: Um die Anzeige von allen ROI-Feldern in einem Diagramm gleichzeitig zu wechseln, klicken Sie auf die Schaltfläche H in der oberen rechten Ecke des Diagramms. Damit können Anwender das Diagramm mit den Minitoolergebnissen exportieren.

Im Allgemeinen kann das ROI-Feld eines Minitools auch als Zeichnungsobjekt betrachtet werden.

  • Um auf den Dialog Objekteigenschaften des Dialogs zuzugreifen, drücken Sie Strg oder Alt, während Sie doppelt auf das ROI-Feld klicken. Der Dialog Objekteigenschaften bietet Zugriff auf die Stileinstellungen des ROI-Felds plus einige technischere Einstellungen, einschließlich des Objektnamens und des Abschnitt vom LabTalk-Skript, der das Hilfsmittel ausführt.
  • Bei einem beliebig geformten ROI-Feld können Sie zweimal einzeln klicken, um den Modus Punkte bearbeiten zu öffnen (Bedienelemente entlang des Rahmens), um die Form weiter anzupassen.

Minitools sind spezialisierte, interaktive Hilfsmittel zum Erzeugen von schnellen Ergebnissen. Es gibt drei "Tastenkombinationen", die Ihnen dabei helfen, Ihre Arbeit zu beschleunigen. Einzelheiten finden Sie in dieser Tabelle.
