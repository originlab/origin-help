Stichprobenumfang für Schätzung

Sample-Size-Estimation

Verwenden Sie diese Funktion, um den Stichprobenumfang für ein bestimmtes Fehlermaß bei Konfidenzintervallen für den Mittelwert, die Standardabweichung, die Varianz, den Anteil, die Rate oder den Mittelwert (Poisson) zu schätzen oder um das Fehlermaß auf der Grundlage des Stichprobenumfangs zu schätzen.

Allgemein gesagt können die Werkzeuge für Trennschärfe und Stichprobenumfang auf zwei Arten verwendet werden ­ zur Stichprobenschätzung und zur Berechnung der Trennschärfe. Ersteres kann verwendet werden, um den Umfang einer Stichprobe zu bestimmen, damit eine bestimmte Trennschärfe beim Aufbau eines Versuchs sicher gestellt wird. Letzteres kann verwendet werden, um die Genauigkeit des Ergebnisses zu schätzen, nachdem ein Versuch für eine festgelegte Anzahl von Stichproben durchgeführt wurde.


Um den Stichprobenumfang mit diesem Hilfsmittel zu schätzen:

  1. Wählen Sie Statistik: Trennschärfe und Stichprobenumfang: Stichprobenumfang für Schätzung. Der Dialog sample_sizes_estimation wird geöffnet.
  2. Wählen Sie den Parameter und geben Sie den Wert für den Parameter ein.
  3. Wählen Sie aus der Liste Berechnen die Option Trennschärfe oder Stichprobenumfang und dann Ihre restlichen Optionen.
  4. Nachdem Sie auf OK geklickt haben, wird ein Analysebericht erstellt.
