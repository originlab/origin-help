Was ist eine virtuelle Matrix?

Wenn Sie ein Konturdiagramm oder ein 3D-Oberflächendiagramm in Origin erstellen möchten, gibt es drei grundlegende Datenanordnungen, die verwendet werden können: XYZ-Arbeitsblattspalten, eine Matrix der Z-Werte oder das, was wir als virtuelle Matrix bezeichnen. Anders als die reguläre Matrix unterstützt die virtuelle Matrix nicht äquidistant verteilte X- und Y-Werte, solange sie sortiert sind.

Die virtuelle Matrix geht üblicherweise von einem Arbeitsblatt aus, das folgendermaßen angeordnet ist: (1) eine Zeile mit Y-Koordinatenwerten oben entlang im Arbeitsblatt und (2) eine Spalte mit X-Koordinatenwerten in der ersten Spalte des Arbeitsblatts (oder umgekehrt), plus eine 2D-Anordnung von "Z"-Werten in einem Block, der durch die Überschneidungen von allen X- und Y-Koordinatenpaaren definiert wird.

X- oder Y-Koordinatenwerte können sich in der ersten Zeile oder Spalte des Datenbereichs im Arbeitsblatt befinden. Sie können im Bereich der Spaltenbeschriftungszeilen gespeichert sein oder sie können in einem anderen Bereich des Arbeitsblatt sein, den Sie z. B. mit dem interaktiven Auswahlelement X-Bereich oder Y-Bereich definiert haben.


Wenn Sie Daten aus MS Excel oder womöglich eine instrumentenerzeugte ASCII-Datei importiert haben und Ihre Datenanordnung der Anordnung im obigen Arbeitsblatt ähnelt, dann verfügen die Daten eventuell über eine Form, die sich zum Erstellen einer virtuellen Matrix eignet.

  • Datenlayout: Y-Daten über Spalten oder X (Daten über Spalten).
  • X-Werte/Y-Werte in 1. Zeile in Auswahl oder 1. Spalte in Auswahl, in Spaltenbeschriftungszeile oder durch interaktive Bereichsauswahl.

Verfügbare 3D- und Konturdiagrammtypen aus virtuellen Matrizen

Im Folgenden werden alle Diagrammtypen, die aus einer virtuellen Matrix erstellt werden können, aufgeführt.

  • Farbige 3D-Fülloberfläche
  • 3D Konstante X mit Basis
  • 3D Konstante Y mit Basis
  • 3D-Oberflächendiagramm mit Farbabbildung (und Farbabbildung mit Projektion)
  • 3D-Punktdiagramm mit Graustufenabbildung
  • 3D-Drahtgitter
  • 3D-Drahtoberfläche
  • 3D-Balkendiagramm für Matrix
  • Kontur - Farbfüllung
  • Konturlinien SW
  • Heatmap

3D- und Konturdiagramme aus einer virtuellen Matrix erstellen

Um ein Kontur- oder 3D-Diagramm aus einer virtuellen Matrix zu erstellen:

  1. Wählen Sie das Arbeitsblatt oder einen Bereich der Arbeitsblattzellen aus und gehen Sie dann zum Menü Zeichnen, um Ihren 3D-Diagrammtyp auszuwählen.
  2. Legen Sie im angezeigten Dialog plotvm fest, wie Ihre Arbeitsblattdaten beim Konvertieren in eine virtuelle Matrix behandelt werden sollen.

Siehe im Tutorial Sich überschneidende Oberflächendiagramme aus Arbeitsblättern erstellen die schrittweisen Anweisungen zum Erstellen von 3D-Diagrammen aus virtuellen Matrizen.

Dialogelemente des Dialogs plotvm

Eingabe Legen Sie die Z-Werte zum Zeichnen eines 3D- oder Konturdiagramms fest. Sie können eine vorhandene virtuelle Matrix als Eingabe im Ausklappmenü oder im virtuellen Matrixmanager auswählen.

Hinweis: Wenn Sie ein ganzes Arbeitsblatt auswählen, wird die virtuelle Matrix dynamisch aktualisiert, wenn Sie Spalten/Zeilen im Quellarbeitsblatt hinzufügen bzw. löschen. Wenn Sie einen Arbeitsblattzellenblock auswählen ab einer bestimmten Spalte/Zeile bis zur letzten Spalte, zum Beispiel von der 5. Spalte bis zum Ende, und möchten, dass die virtuelle Matrix dynamisch aktualisiert wird, klicken Sie mit der rechten Maustaste auf die dreieckige Schaltfläche rechts von Eingabe und wählen Sie Dynamische Enden.
Datenlayout Legen Sie fest, wie die X- und Y-Werte für den Graphen angegeben werden. Verfügbare Optionen sind:
  • Y über Spalten
    Verwendet die Werte in einer Zeile als Y-Werte und die Werte in einer Spalte als X-Werte.
  • X über Spalten
    Verwendet die Werte in einer Zeile als X-Werte und die Werte in einer Spalte als Y-Werte.
Y-Werte in/X-Werte in Die zwei Auswahllisten werden verwendet, um den X- und Y-Datenbereich beim Erstellen der virtuellen Matrix festzulegen. Die Abfolge dieser beiden ändert sich dynamisch in Abhängigkeit von der Auswahl des Datenlayouts, und die obere Liste wird verwendet, um eine Zeile festzulegen, während die untere zum Festlegen einer Spalte dient. Die verfügbaren Optionen entsprechen den folgenden:
  • Kein
    Verwenden Sie den Zeilenindex oder Spaltenindex.
  • 1. Zeile/Spalte in der Auswahl
    Verwenden Sie die Werte der ersten Zeile/Spalte in der aktuellen Auswahl.
  • 1. Zeile/Spalte in Arbeitsblatt
    Verwenden Sie die Werte der ersten Zeile/Spalte im aktuellen Arbeitsblatt.
  • Spaltenbeschriftung
    Diese Option ist nur dann verfügbar, wenn eine Zelle festgelegt werden muss. Sie verwendet die Werte der Spaltenbeschriftungen im Arbeitsblatt. Mit dieser Option können Sie eine Spaltenbeschriftungszeile aus der Auswahlliste Spaltenbeschriftung auswählen. Bitte beachten Sie, dass, sollte nur ein Teilbereich des Arbeitsblatts als Eingabedaten ausgewählt sein, dieses Hilfsmittel dann automatisch den entsprechenden Teil der Werte der Spaltenbeschriftung verwendet.
  • Benutzerdefiniert
    Mit dieser Option wird die manuelle Datenauswahl mit dem Feld X-Bereich (oder Y-Bereich) möglich. Bitte beachten Sie, dass die X(oder Y)-Werte nur aus dem gleichen Arbeitsblatt ausgewählt werden können.
Y-Daten formatieren/X-Daten formatieren Legen Sie das Datenformat der Y/X-Daten als Numerisch (Standard), Text, Zeit oder Datum fest. Aktivieren Sie dieses Kontrollkästchen und klicken Sie auf die Schaltfläche ... rechts vom Textfeld, um den Dialog Daten formatieren zu öffnen, und wählen Sie das bzw. die auf die Y/X-Daten anzuwendende Format und Anzeige.

Wenn Y-Werte in/X-Werte in auf Keine gesetzt ist, ist dieses Kontrollkästchen deaktiviert.
X-Titel/Y-Titel/Z-Titel Legen Sie die Achsentitel für die X-, Y- bzw. Z-Achse fest. Deaktivieren Sie das Kontrollkästchen am Ende des Bedienelements. Sie können auf die Schaltfläche mit dem Pfeil klicken, um die Informationen des Quellblatts als Titel im Ausklappmenü auszuwählen.

Die Auswahl eines Eintrags im Ausklappmenü fügt die entsprechende Notation in das Feld ein. Die tatsächlichen Titel werden unter den Bedienelementen als blauer Hinweis gezeigt:

  • LG: Langname und Einheiten: Die Auswahl fügt die Notation %(%@H, 1, LG) ein.
  • L: Langname: Die Auswahl fügt die Notation %(%@H, 1, L) ein.
  • U: Einheiten: Die Auswahl fügt die Notation %(%@H, 1, U) ein.
  • C: Kommentare: Die Auswahl fügt die Notation %(%@H, 1, C) ein.
  • P1: Erster Parameter: Die Auswahl fügt die Notation %(%@H, 1, P1) ein.
  • D1: Erster Anwenderparameter: Die Auswahl fügt die Notation %(%@H, 1, D1) ein.
  • LH1: Erster Anwenderparametername: Die Auswahl fügt die Notation %(%@H, @LH1) ein.
  • SN: Blattname: Die Auswahl fügt die Notation %(%@H, @SN) ein.
  • PNL: Mappenlangname: Die Auswahl fügt die Notation %(%@H, @PNL) ein.
  • Aus Arbeitsblatt auswählen: Die Auswahl öffnet einen Dialog, über den Sie auf eine Zelle im Arbeitsblatt klicken können. Die Notation für die ausgewählte Zelle wird gezeigt. Wenn Sie zum Beispiel eine Zelle (1. Spalte, 3. Zeile) im Quellblatt der virtuellen Matrix auswählen, lautet die Notation %(%@H, 1, 3). Wenn Sie eine Zelle auswählen, die sich nicht im Quellblatt befindet, erstellen wir %([Book1]Sheet1, 1, 3, W) wie im Dialog Variablen einfügen.
Name der virtuellen Matrix Legen Sie den Kurznamen des zu erstellenden Objekts der virtuellen Matrix fest. Alle virtuellen Matrizen, die während des Zeichnens erstellt wurden, werden im Virtuellenl Matrixmanager aufgeführt und verwaltet.
