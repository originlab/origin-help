Benutzerdefinierte Tastenfolgen

Custom-Chorded-Keys

  • Tastenfolgen


Aktuelle Standard-Tastenkombinationen mit STRG haben alle Zahlen und Buchstaben auf der Tastatur erschöpft. Um weitere Kombinationen zu unterstützen, wurden in Origin 2025b die Tastenfolgen eingeführt.

Wenn Sie die STRG-Taste gedrückt halten und dann die Präfixtaste ”\” drücken, wartet Origin auf die zweite Taste der Folge (Hinweis in der Statusleiste unten) und ein Nachrichtenfeld wird angezeigt mit einer Liste der Tastenfolgen, die vordefiniert sind. Sie können die gewünschte zweite Taste wählen, um die gewünschte Aktion auszuführen.

Wenn Sie mit der Maus während des Wartemodus‘ für die zweite Taste irgendwohin klicken, wird das Warten abgebrochen.

Tastenfolgen hinzufügen und benutzerdefiniert anpassen

Sowohl die Präfixtaste als auch die zweite Taste können im Dialog Tastenfolgen geändert werden.

Um den Dialog Tastenfolgen zu öffnen:

  • Wählen Sie im Menü Einstellungen: Tastenfolgen.

Kontext

Bestimmen Sie den/die für die aktuelle Tastenkombination verfügbare(n) Fenstertyp(en). Wenn Sie ein Menüelement wählen, das für alle Fenstertypen verfügbar ist, wird der Kontext auf Beliebig gesetzt. Falls nicht, wird Kontext auf den aktiven Fenstertyp gesetzt.

  • Beliebig
Wenn das von Ihnen gewählte Menüelement für alle Fenstertypen verfügbar ist, z. B. Fenster: Anordnen, sollte für Kontext die Option Beliebig gewählt sein. Wenn das Menü zu mehreren Fenstertypen gehört anstatt zu allen Typen, z. B. Bearbeiten: Exportrand festlegen ... (verfügbar für sowohl Diagramm- als auch Layoutfenstertypen), wird der Kontext auf den aktiven Fenstertyp gesetzt.

Präfix

Die Standardpräfixtaste ist der Backslash “\”. Sie kann geändert werden in die folgenden Tasten:

Slash (/) Backslash (\)
Rücktaste Taste 1 (!)
Semikolon (;) Apostroph (')
Komma (,) Punkt (.)

Neben diesem Dialog können Sie auch die Datei KeyChords.ini im Anwenderdateiordner bearbeiten.

Die 2. Taste

Die zweite Taste kann dann eine beliebige andere Taste auf der Tastatur sein, einschließlich die Pfeiltasten, die Tasten Enter, Leerzeichen, Entfernen etc. Origin wird mit einem vordefinierten Satz Tasten ausgeliefert, die Sie im Listenfeld des Dialogs Tastenfolgen finden können.

Sie können alle bearbeitet und gelöscht werden, und Sie können auch neue Tasten hinzufügen.

  • Menü-ID
    Um eine Menü-ID zu suchen:
    Drücken Sie STRG+SHIFT und wählen Sie die gewünschte Menüoption. Die Menü-ID (und ggf. das Skript) wird im Skriptfenster eingegeben, aber nicht ausgeführt. Falls Quellcode für die Menüoption existiert (LabTalk oder Origin C), wird der Code Builder mit der Quelldatei geöffnet.
    Er kann 0 sein, wenn die Aktion vom LabTalk-Skript durchgeführt wird.<.li>
  • Aktion
    Erklärungstext der Tastenkombination. Wenn die Tastenkombination ein Menü oder eine Schaltfläche auslöst, gibt der Text das Menüelement bzw. die Symbolleistenschaltfläche an. Andernfalls zeigt der Text den LabTalk-Befehl (bei mehreren Zeilen wird die erste angezeigt), den die Tastenkombi ausführt, oder die Wirkung, die durch die Tastenkombi erzielt wird.
  • Kontext
    Wenn Kontext auf Alle gesetzt ist, werden alle definierten Tastenfolgen aufgeführt. Die Kontextspalte wird dann bis zum Letzten gezeigt, um Sie über die verfügbaren Fenstertypen für die Tastenkombinationen zu informieren.


Klicken Sie mit der rechten Maustaste auf ein Listenelement. Wenn Sie mit der rechten Maustaste auf eine leere Fläche klicken, wird eine Liste mit Kontextmenüs aufgerufen.

Auf Menüelement Auf leere Fläche
Auswählen Wählen Sie ein Menü oder eine Symbolleistenschaltfläche, um eine Tastenkombination für ihm bzw. sie festzulegen.

Die Menü-ID und Aktionsbeschreibung werden im Dialog Tastenfolgen wiedergegeben. Drücken Sie die Esc-Taste, um den Auswahlmodus zu verlassen. Siehe Einzelheiten unter Eine neue Tastenfolge hinzufügen.
2. Taste erweitert bearbeiten… Klicken Sie doppelt auf die Zelle der 2. Taste oder klicken Sie mit der rechten Maustaste auf sie und wählen Sie 2. Taste erweitert bearbeiten im Kontextmenü, um einen kleinen Dialog zum Auswählen der 2. Taste zu öffnen.

Geben Sie die 2. Taste ein oder wählen Sie sie in der Auswahlliste.

LabTalk-Befehl bearbeiten		 Fügen Sie eine Tastenkombination für den/die LabTalk-Befehl(e) hinzu. Der Befehl wird in der Spalte Aktion gezeigt. Falls es mehrere Befehlszeilen gibt, wird nur die erste Zeile gezeigt.

Hinzufügen

Fügen Sie eine neue Tastenfolge hinzu. Einzelheiten lesen Sie bitte im Abschnitt unten.

Eine neue Tastenfolge hinzufügen

  1. Drücken Sie die Strg-Taste und klicken Sie auf das gewünschte Menüelement oder eine Symbolleistenschaltfläche, um den Dialog Tastenfolgen zu öffnen. Die entsprechende Menü-ID und Beschreibung der Aktion werden automatisch zur Dialogliste hinzugefügt. Die Zelle für die 2. Taste befindet sich im direkten Bearbeitungsmodus und wartet darauf, dass Sie eine zweite Taste für die Kombi hinzufügen.
  2. Wählen Sie im Menü Einstellungen: Tastenfolgen oder drücken Sie Strg+Alt+K, um den Dialog Tastenfolgen zu öffnen.
    • Klicken Sie mit der rechten Maustaste auf die leere Fläche innerhalb des Listenfelds und wählen Sie Auswählen. Der Dialog wird verborgen und ermöglicht Ihnen, ein Menüelement oder eine Symbolleistenschaltfläche zu wählen, um für diese/s eine Tastenkombination hinzuzufügen. Danach kehren Sie zum Dialog Tastenfolgen zurück und die ausgewählte Menü-ID und der Text zur Aktion werden automatisch zum Listenfeld hinzugefügt. Sie können die 2. Taste direkt in die Spalte 2. Taste eingeben.
    • Klicken Sie auf die Schaltfläche Hinzufügen. Der Dialog wird verborgen und ermöglicht Ihnen, ein Menüelement oder eine Symbolleistenschaltfläche zu wählen, um für diese/s Tastenkombination hinzuzufügen. Danach kehren Sie zum Dialog Tastenfolgen zurück und die Menü-ID und der Text zur Aktion werden automatisch zum Listenfeld hinzugefügt. Geben Sie die 2. Taste direkt in die Zelle der 2. Taste ein, um die Definition der Tastenkombination zu vervollständigen.
