Datenanforderungen

Wählen Sie mindestens eine Spalte aus, um ein Wilkinson-Punktdiagramm oder ein Histogramm mit Punkten zu zeichnen.

Diagramm erstellen

Um ein Punktdiagramm zu erstellen:

  1. Markieren Sie mindestens eine Arbeitsblattspalte aus (oder einen Bereich aus mindestens einer Spalte).
  2. Wählen Sie Zeichnen > Statistisch: Punktdiagramm.

Vorlage

statdot.otpu (installiert im Origin-Programmordner)

Notizen

Wenn Sie mehrere Eingabespalten auswählen, wird jede Spalte als ein Punktdiagramm gezeichnet, bei dem die Punkte von der Basislinie aus angeordnet werden. Diese Punktdiagramme werden auf der Y-Achse gestapelt.

Weitere Einzelheiten finden Sie auf der Seite Punktdiagramm erstellen.
