Datenanforderungen

Wählen Sie Temperatur und Geschwindigkeitskonstante als XY-Spalte, um ein Arrhenius-Diagramm zu erstellen.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen > Mehrere Felder, mehrere Achsen: Arrhenius im Hauptmenü.

Hinweis: Das Auswählen dieses Menüs fügt zwei weitere Spalten zum Quellblatt hinzu, eine Spalte ist 1/(first column), die zweite ist ln(2nd column). Nehmen Sie diese zwei Spalten als XY, um ein Punktdiagramm im ersten Layer zu zeichnen. Fügen Sie den zweiten Layer mit X mit benutzerdefinierter Formel hinzu.

Vorlage

ARRHENIUS.otpu (im Origin-Programmordner installiert)

Hinweise

Dieses Diagramm enthält zwei Layer, von denen einer ein Linien-Punktdiagramm mit neu berechneten Spalten zeichnet und der andere die obere X-Achse mit benutzerdefinierter Formel zeigt.
