Datenanforderungen

Wählen Sie mindestens XY-Spalten aus, um ein Diagramm mit doppeltem X zu erstellen, wobei die zweite X-Achse funktional auf die erste abgebildet wird.

Ideal zum Anzeigen von Daten in verschiedenen Einheiten, wie Temperatur (°C/°F/K), Wellenlänge und Wellenanzahl.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen > Mehrere Felder, mehrere Achsen: Doppeltes X mit benutzerdefinierter Formel im Hauptmenü.

Es wird ein Dialog angezeigt, mit dem Sie die sekundäre Achse festlegen können:

Es gibt zwei Methoden, um auf funktional auf die doppelten X-Achsen abzubilden: Unabhängige Hilfsstriche und Hilfsstriche teilen.

  • Unabhängige Hilfsstriche: Das Auswählen dieses Typs ermöglicht Ihnen, die Direkte Formel und Inverse Formel einzugeben, um die sekundäre X-Achse in einem anderen Layer ohne Y-Achse darzustellen. Die Hilfsstrichsposition wird durch die sekundäre Achse selbst entschieden. Sie können festlegen, ob diese beiden Layer den Y-Bereich teilen.
    Weitere Einzelheiten finden Sie unter dem Skalierungstyp Benutzerdefinierte Formel auf der Registerkarte Skalierung im Dialog Achsen.
  • Hilfsstriche teilen: Wählen Sie diesen Typ, um die geteilten Hilfsstriche auf der sekundären X-Achse des aktuellen Layers zu zeigen. Die Hilfsstrichsbeschriftungen werden durch die von Ihnen eingegebene Formel entschieden.

Vorlage

dxcform.otpu (im Origin-Programmordner installiert)

Hinweise

Dieses Diagramm enthält zwei Layer, wenn Sie Funktional abgebildet - unabhängige Hilfsstriche als Typ Doppeltes X auswählen. Die sekundäre X-Achse wird zum zweiten Layer hinzugefügt. Wenn Sie Funktional abgebildet - Hilfsstriche teilen ausgewählt haben, wird nur ein Layer mit unterer und oberer X-Achse gezeichnet, die die Hilfsstrichsposition teilen und verschiedene Hilfsstrichsbeschriftungen je nach festgelegter Formel zeigen.
