Datenanforderungen

Wählen Sie mindestens eine Spalte aus, um einmalige Werte zu zählen, und erstellen Sie ein Säulen-/Balken-/Kreis-/Ring-/Sunburst-Diagramm.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie im Menü Zeichnen > Balken, Kreis, Fläche: Häufigkeitsdiagramm. Der Dialog plot_frequency wird angezeigt.

Der Dialog "plot_frequency"

Eingabe Wählen Sie mindestens eine Spalte als Quelldaten, um das Häufigkeitsdiagramm zu berechnen und zu zeichnen.
Häufigkeiten Wählen Sie eine oder mehrere Spalten für die Häufigkeitsdaten als optionale Eingabe.

Wenn Mehrere Daten auf Separate Grafiken gesetzt ist, ermöglicht diese Option, dass mehrere Spalten als Häufigkeitsdaten ausgewählt werden können. Die Anzahl der Spalten könnte 1 oder gleich der Anzahl der Eingabespalten sein.

  • Wenn Sie nur eine Spalte für diese Option ausgewählt haben, dann nimmt jede Eingabespalte diese Spalte als Häufigkeit.
  • Wenn Sie mehrere Spalten für die Option ausgewählt haben, nimmt jede Eingabespalte eine ausgewählte Spalte der Reihe nach als Häufigkeitsdaten . Der Y-Wert ist die Summe der Häufigkeitsdaten nach Eingabekategorie.

Wenn Mehrere Daten auf Geschachtelte Anzahl in gleicher Grafik gesetzt ist, kann für diese Option nur eine Spalte gewählt werden.
Mehrere Daten Legen Sie fest, wie die ausgewählten Spalten für mehrere Daten gezeichnet werden sollen.
  • Separate Grafiken: Zeichnen Sie mehrere Eingabespalten in separate Grafiken.
  • Geschachtelte Anzahl in gleicher Grafik: Zählen Sie die geschachtelten Anzahlen der letzten Eingabespalte basierend auf allen ausgewählten Spalten und zeichnen Sie sie dann auf der gleichen Grafik.
Felder teilen nach Wenn dieses Kontrollkästchen aktiviert ist, können Sie eine andere Gruppierungsspalte auswählen, um die horizontalen Felder zu aktivieren, die ein Häufigkeitendiagramm für jeden Gruppierungswert separat zeigen. Felder umbrechen wird aktiviert, Hilfsstriche und Beschriftungen werden standardmäßig auf abwechselnden Seiten gezeigt.

Wenn mehrere Spalten ausgewählt sind, wird nur eine Beschriftungszeile für die Feldinformation oberhalb gezeigt. Die Zeichenkette des Feldbanners schließt alle Faktoren ein und ist durch Komma getrennt. Falls Mehrere Daten auf Geschachtelte Anzahl in gleicher Grafik gesetzt ist, wird die Option verborgen.
Seiten teilen nach Wenn dieses Kontrollkästchen aktiviert ist, können Sie eine andere Gruppierungsspalte auswählen, um die Eingabedaten zu teilen und das Häufigkeitendiagramm auf verschiedenen Diagrammseiten zu erstellen. Jede Seite zeichnet nur die Spalten innerhalb der für die Seite relevanten Gruppe. Die seitenrelevanten Gruppeninformationen werden im Layertitel gezeigt, getrennt durch Komma, falls es mehrere Faktoren gibt. Das Berichtsdiagrammblatt führt alle Seiten auf.
Diagrammtyp Wählen Sie einen Diagrammtyp, um die Häufigkeiten der Quelldaten zu präsentieren.
  • Spalte
  • Balken
  • Kreis
  • Ring: Nur für mehrere Spalten und wenn Mehrere Daten auf Separate Grafiken gesetzt ist.
  • Sunburst: Nur für mehrere Eingabespalten und wenn Mehrere Daten auf Geschachtelte Anzahl in gleicher Grafik gesetzt ist.
  • Sunburst mit Loch: Nur für mehrere Eingabespalten und wenn Mehrere Daten auf Geschachtelte Anzahl in gleicher Grafik gesetzt ist.
Y teilen Diese Option wird nur gezeigt, wenn mehrere Spalten der Quelldaten für das Feld Eingabe ausgewählt wurden und Mehrere Daten auf Separate Grafiken für den Säulen- oder Balkendiagrammtyp gesetzt wurde. Sie wird verwendet, damit verschiedene Grafiken sich die gleiche Y-Skalierung teilen.
Stapeln Das Bedienelement ist nur verfügbar, wenn es mehrere Eingabedaten gibt, Mehrere Daten auf Geschachtelte Anzahl in gleicher Grafik und Diagrammtyp auf Säulen oder Balken gesetzt wurde.

Legen Sie fest, wie die geschachtelte Anzahl als Säulen/Balken gezeichnet werden soll. Es gibt drei Möglichkeiten, die Anzahl als Säulen/Balken zu präsentieren:

  • Nebeneinander
  • Gestapelt (Letzte Säule)
  • Gestapelt, 100% (Letzte Säule)
Diagrammmvorlage Diese Option zeigt die Standardvorlage, um unterschiedliche Diagrammtypen zu erstellen. Sie können das Kontrollkästchen Auto deaktivieren, um eine andere Vorlage zu wählen, wie eine benutzerdefinierte Vorlage.
Ausgabezeichnungsdaten Legen Sie fest, in welches Blatt die Zeichnungsdaten ausgegeben werden.
Diagramme ausgeben Legen Sie fest, in welches Blatt die Ergebnisdiagramme ausgegeben werden.
