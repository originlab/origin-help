Brief Description

Gram-Charlier-Peakfunktion zur Verwendung in der Chromatografie.

Sample Curve

Parameter

Number: 6

Names: y0, xc, A, w, a3, a4

Meanings: y0 = offset, xc = center, A=area, w=half width, a3 = unknown, a4 = unknown

Lower Bounds: w > 0.0

Upper Bounds: none

Script Access

nlf_gcas(x,y0,xc,A,w,a3,a4)
