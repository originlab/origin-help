Einführung

Dieses Hilfsmittel kann verwendet werden, um den grafischen Zusammenfassungsbericht der Eingabedaten mit Gruppierungsinformationen zu erstellen. Der Bericht enthält Histogramme, Boxdiagramme, Kondenzintervalldiagramme und Ergebnistabellen mit statistischen Informationen.

Grafische Zusammenfassung durchführen

  • Wählen Sie die Spalten der Eingabedaten aus, für die Sie eine einfache statistische Analyse durchführen möchten.
  • Wählen Sie Statistik > Deskriptive Statistik: Grafische Zusammenfassung im Menü, um das Hilfsmittel zu öffnen.
  • Wählen Sie die Gruppierungsspalte(n) für die Option Gruppe(n), geben Sie einen Wert im Feld Konfidenzniveau (%) ein und wählen Sie eine gewünschte Anordnung in der Liste Diagrammanordnung aus.
  • Klicken Sie auf OK, um eine deskriptive Statistik durchzuführen und die Diagramme zu erstellen.

Bedienelemente des Dialogs

Eingabe Wählen Sie die Eingabespalte(n).
Gruppen Wählen Sie (eine) Gruppierungsspalte(n), um die Eingabedaten in mehrere Gruppen aufzuteilen.
Kondenzintervall (%) Legen Sie das Konfidenzniveau des Konfidenzintervalls fest.
Diagrammanordnung Legen Sie fest, wie die Statistikdiagramme angeordnet werden.
  • Separate Grafiken: Jede Variable für jede Gruppe wird in eine separate Grafik gezeichnet. Jede Grafik enthält ein Histogramm mit Verteilungskurve, ein Konfidenzintervalldiagramm und ein Wahrscheinlichkeitsdiagramm mit drei Ergebnistabellen "Deskriptive Statistik", "##% Konfidenzintervalle" und "Test auf Normalverteilung". Der Grafiktitel zeigt die Variable und die Gruppeninformation.
  • Pro Variable: Jede Variable wird in eine separate Grafik gezeichnet. Die Gruppen werden in verschiedene vertikale Felder gezeichnet. Jede Grafik enthält Histogramme mit Verteilungskurve, Boxdiagramme mit Ausreißern, Konfidenzintervalldiagramme und eine Tabelle, die die Ergebnisse nach Zeile zusammenfasst und die Statistiken für jede Gruppe zeigt.
  • Pro Gruppe: Jede Gruppe wird in eine separate Grafik gezeichnet. Die Variablen werden in verschiedene vertikale Felder gezeichnet. Jede Grafik enthält Histogramme mit Verteilungskurve, Boxdiagramme mit Ausreißern, Konfidenzintervalldiagramme und eine Tabelle, die die Ergebnisse nach Zeile zusammenfasst und die Statistiken für jede Variable zeigt.
  • Alle in einer: Alle Variablen und alle Gruppe werden in einer Grafik gezeichnet. Jeder Teildatensatz (Gruppe*Variable) wird in verschiedene vertikale Felder gezeichnet. Jede Grafik enthält Histogramme mit Verteilungskurve, Boxdiagramme mit Ausreißern, Konfidenzintervalldiagramme und eine Tabelle, die die Ergebnisse nach Zeile zusammenfasst und die Statistiken für jeden Teildatensatz zeigt.
Ausgabezeichnungsdaten Legen Sie fest, wo die Zeichnungsdaten ausgegeben werden sollen.
Diagramme ausgeben Legen Sie fest, wo die Ergebnisdiagramme ausgegeben wird.
