Datenanforderungen

Wählen Sie mindestens eine Spalte als Eingabe aus und bereiten Sie mindestens eine Gruppenspalte vor, um gruppierte Punktdiagramme zu erstellen.

Diagramm erstellen

Um ein gruppiertes Punktdiagramm zu erstellen:

  1. Markieren Sie mindestens eine Spalte aus (oder einen Bereich aus mindestens einer Spalte).
  2. Wählen Sie Zeichnen > Kategorial : Gruppierte Punktdiagramme, um den Dialog plot_gdot zu öffnen. Wählen Sie mindestens eine Gruppierungsspalte im Feld Gruppe.

Vorlage

statdot.otpu (installiert im Origin-Programmordner)

Notizen

Wenn Sie mehrere Eingabespalten wählen, wird jede Spalte als ein gruppiertes Punktdiagramm gezeichnet, bei dem die Punkte von der Basislinie aus angeordnet sind. Diese gruppierten Punktdiagramm werden auf der Y-Achse gestapelt.

Weitere Einzelheiten finden Sie auf der Seite Punktdiagramm erstellen.
