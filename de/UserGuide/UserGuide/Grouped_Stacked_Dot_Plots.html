Datenanforderungen

Wählen Sie mindestens eine Spalte als Eingabe aus und bereiten Sie mindestens eine Gruppenspalte vor, um gruppierte gestapelte Punktdiagramme zu erstellen.

Diagramm erstellen

Um ein gruppiertes Punktdiagramm zu erstellen:

  1. Markieren Sie mindestens eine Spalte aus (oder einen Bereich aus mindestens einer Spalte).
  2. Wählen Sie Zeichnen > Kategorial: Gruppierte gestapelte Punktdiagramme, um den Dialog plot_gdot zu öffnen. Wählen Sie mindestens eine Gruppierungsspalte im Feld Gruppe.

Vorlage

StackedDot.otpu (installiert im Origin-Programmordner)

Notizen

Wenn Sie mehrere Eingabespalten auswählen, wird jede Spalte als ein gruppiertes gestapeltes Punktdiagramm gezeichnet, bei dem die Punkte von der Basislinie aus angeordnet und diese gruppierten gestapelten Diagramme auf der Y-Achse gestapelt werden.

Wenn es nur eine Gruppierungsspalte gibt, können Sie das Element Stapeln verwenden, um zu entscheiden, ob Sie ein gruppiertes oder ein gestapeltes Punktdiagramm zeichnen möchten. Weitere Einzelheiten finden Sie auf der Seite Punktdiagramm erstellen.
