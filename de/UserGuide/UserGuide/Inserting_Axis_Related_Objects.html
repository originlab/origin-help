Achsenbezogene Objekte einfügen

Insert-Axis-Related-Objects

Sie können den Dialog Achsen immer öffnen, um Objekte zu den Achsen hinzuzufügen, wie gerade Linien, Referenzlinien etc. Diese Seite zeigt Ihnen, wie Sie diese achsenbezogenen Objekte schnell und intuitiv hinzufügen.

Eine vertikale/horizontale gerade Linie hinzufügen

Wählen Sie Einfügen: Gerade Linie..., um den Dialog Gerade Linie: addline zu öffnen.

Sie können eine vertikale oder horizontale gerade Linie mit Beschriftungen an spezifischen Positionen auf der X- oder Y-Achse einfügen.

Sie können den Zweig Linienformat erweitern, um den Linienstil benutzerdefiniert anzupassen. Sie können über das Anzeigeformat und die Beschriftungspositionen entscheiden.

Wenn Auswählbar und Verschiebbar aktiviert sind, kann die Linie per Drag&Drop an andere Stellen der Achse gezogen werden und die Beschriftung wird automatisch aktualisiert.

Um Referenzlinien hinzuzufügen:

Wählen Sie Einfügen: Referenzlinie..., um den Dialog Referenzlinien einfügen zu öffnen.

Dieser Dialog kann auch über Klicken auf die Achse geöffnet werden, um dann auf die Schaltfläche Referenzlinie hinzufügen zu klicken:

Achse auswählen

Sie können eine Achse wählen, zu der Sie die Referenzlinien hinzufügen.

Positionswerte eingeben

Sie können mehrere durch Leerzeichen getrennte Werte in die Felder Bei X= oder Bei Y = eingeben, um mehrere Referenzlinien hinzuzufügen.

Durch Klicken auf die Schaltfläche erweitern Sie die Liste der Statistikzeichenketten, um einen statistischen Wert einzugeben.

Detaillierten Dialog der Referenzlinien öffnen

Das Klicken auf die Schaltfläche Mehr... öffnet den Dialog Referenzlinien, mit dem Sie die hinzugefügten Referenzlinien weiter individuell gestalten können.

Hinweis: Wenn Sie eine Referenzlinie hinzugefügt haben, können Sie auf sie klicken und die Minisymbolleiste zur weiteren benutzerdefinierten Anpassung verwenden.
