Was ist der Life-Science-Modus?

Der Life-Science-Modus wird in Origin 2026b eingeführt, um den Schwerpunkt auf Arbeitsabläufe im Bereich Life Science zu legen. Dieser Modus vereinfacht die Origin-Bedienoberfläche, indem die meisten der Symbolleistenschaltflächen zum Zeichnen und die erweiterten Analysemenüs verborgen werden und das Menü Zeichnen neu strukturiert wird, um statistische und biologische Diagramme und Analysen leichter zugänglich zu machen. Eine besonderes Arbeitsblatt "Datentabelle" vereinfacht das Erstellen von Statistikdiagrammen und Analyseroutinen. Häufige verwendete Life-Science-Dateiformate wie .XML können direkt per Drag&Drop importiert werden. Dieser Modus installiert auch eine Sammlung von beliebten Apps wie DOE vor.

Um Ihnen beim Start des Life-Science-Modus zu helfen, zeigt Origin jedes Mal einen Start an, wenn Sie Origin öffnen oder ein neues Projekt erstellen. Beim Öffnen des Dialogs ist die Registerkarte Beispiele aktiv. Sie bietet eine Reihe von typischen Beispielen inkl. Beschreibungen, die Sie dabei unterstützen, Ihre Analyseroutinen mit Leichtigkeit zu meistern.

Da der Life-Science-Modus nur einige einfache Menüs und Schaltflächen behält, nutzen Sie bitte das Suchfeld, wenn Sie ein Hilfsmittel nicht auf der Bedienoberfläche finden können, um es zu suchen und auszuführen.

Life-Science-Modus installieren

Während der Origin-Installation wählen Sie die Option Life-Science-Bedienoberfläche auf der Seite "Modus der grafischen Bedienoberfläche (GUI) auswählen". Lesen Sie in der Origin-Installationsanleitung eine schrittweise Anleitung.

Wenn die Life-Science-Bedienoberfläche ausgewählt ist, startet Origin standardmäßig im Life-Science-Modus.


Hinweise:

Wenn Sie die Life-Science-Bedienoberfläche aktiviert ist, ist eine der vorinstallierten Apps Design of Experiments (DOE). Diese App erfordert, dass die R-Software in der Version 4.4.1 VOR OriginPro installiert wurde.

Spalte

Dieser Tabellentyp folgt der Struktur YYY...: mehrere Y-Spalten und keine X-Spalte. Jede Y-Spalte stellt eine unabhängige Gruppe dar, und Replikationen werden in jeder Spalte angeordnet. Er kann eine optionale Spalte Zeilentitel (Textspalte zum Beschriften von Replikationen) enthalten.

Gruppennamen Geben Sie mit Semikolon getrennte Namen für Gruppen (Y-Spalten) ein.
Zeilentitel Aktivieren Sie dieses Kontrollkästchen, um eine Textspalte für Zeilenbeschriftungen hinzuzufügen.


Spaltentabellen werden hauptsächlich für Rohdaten von einfachen ANOVA und anderen Analysen verwendet, die unabhängige Gruppen gemessen an der gleichen Variable vergleichen (z. B. t-Tests, nicht-parametrische Tests).

Gruppiert

Dieser Tabellentyp folgt der Struktur Zeilentitel + Y-Spalten: eine Titelspalte für Zeilenbeschriftungen, gefolgt von mehreren Y-Spalten, die in Gruppen organisiert sind. Jede Gruppe stellt eine Stufe des Faktors A dar und die Y-Spalten innerhalb jeder Gruppe stellen die Stufen des Faktors B dar.

Optional können Sie Teilgruppen aktivieren, um den dritten Faktor für die dreifache ANOVA hinzuzufügen. Teilgruppen teilen jede Gruppe (Faktor A) in zusätzliche Stufen, die Faktor B darstellen, während die Y-Spalten innerhalb jeder Teilgruppe Faktor C darstellen.

Beachten Sie, dass alle Faktoren (einfache und zweifache) gekreuzt sind (alle Kombinationen präsent).

Anzahl der Gruppen Wählen Sie oder geben Sie die Anzahl der zu vergleichenden Gruppen über zwei (oder drei) kategoriale Dimensionen ein.

Hinweis: Sie können Gruppen jederzeit nach Erstellen der Tabelle hinzufügen oder entfernen.

  • Um eine oder mehrere Gruppen hinzuzufügen, klicken Sie auf das Symbol eines Tabellentyps und wählen Sie im Menü Gruppen hinzufügen. Einzelheiten lesen Sie bitte in diesem Abschnitt.
  • Um eine neue Gruppe vor einer vorhandenen Gruppe einzufügen, klicken Sie mit der rechten Maustaste auf die Titelleiste des Datensatzes und wählen Sie Einfügen.
  • Um eine Gruppe zu löschen, klicken Sie mit der rechten Maustaste auf die Titelleiste der Gruppe und wählen Sie Löschen.
Gruppennamen Geben Sie Namen der Gruppen mit Semikolon getrennt ein.
Anzahl der Teilgruppen Bei der dreifachen ANOVA wählen Sie oder geben Sie unter Anzahl der Teilgruppen einen Wert ein, der nicht 0 ist, um den dritten Faktor hinzuzufügen. Teilgruppen teilen jede Gruppe in zusätzliche Stufen, die Faktor B darstellen, während die Y-Spalten innerhalb jeder Teilgruppe Faktor C darstellen. Dies erstellt einen gekreuzten Versuchsplan mit drei Faktoren, wobei Faktor A (Gruppen), Faktor B (Teilgruppen) und Faktor C (Y-Spalten) alle miteinander gekreuzt sind.
Werte pro Gruppe replizieren Wählen Sie oder geben Sie die Anzahl der replizierten Y-Spalten pro Gruppe ein. Geben Sie 1 für keine Replikationen ein.

Zeilentitel Aktivieren Sie dieses Kontrollkästchen, um eine Textspalte für Zeilenbeschriftungen hinzuzufügen.


Gruppierte Tabellen werden hauptsächlich für zweifache ANOVA und Analysen mit zwei unabhängigen kategorialen Variablen (Faktoren) verwendet. Jeder Faktor hat zwei oder mehr Stufen.

Kontingenz

Dieser Tabellentyp folgt einer zweifachen Kontingenzstruktur: eine Textspalte für Zeilenkategoriebeschriftungen, gepaart mit mehreren numerischen Spalten, die Häufigkeitsanzahlen enthalten. Die erste Textspalte enthält Beschriftungen für die erste kategoriale Variable. Die folgenden numerischen Spalten enthalten ganzzahlige Anzahlen für jede Stufe der zweiten kategorialen Variable.

Spaltenkategorien Geben Sie mit Semikolon getrennte Beschriftungen für die zweite kategoriale Variable ein.


Die Kontingenztabellen sind für kategoriale Datenanalysen, wie dem Chi-Quadrat-Test und dem exakten Test nach Fisher, entwickelt worden.

Lebensdauer

Dieser Tabellentyp folgt der Struktur Zeit+ Status: eine X-Spalte für die abgelaufene Zeit, gepaart mit einer oder mehreren Y-Spalten für den Ereignisstatus. Die X-Spalte enthält die "abgelaufene Zeit" ab Beginn der Beobachtung bis zum Ereignis oder zur Zensierung. Die Y-Spalten enthalten den "Status", der angibt, ob ein Ereignis eingetreten ist: typiscerweise ist 0 = Zensiert (kein Ereignis/zurückgezogen), 1 = Ereignis (Tod/Fehlschlag/festgelegtes Ergebnis ist aufgetreten). Eine optionale Spalte für Zeilentitel kann für einzelne Subjektbeschriftungen hinzugefügt werden.

Anzahl der Gruppen Wählen Sie oder geben Sie die Anzahl der zu vergleichenden Gruppen ein. Eine Y-Spalte pro Gruppe

Zusammenfassungsdaten zeichnen

Wählen Sie das Zeichnen der statistischen Zusammenfassungsdaten entweder als Punkt-, Linien-, Punkt-Linien- oder Säulendiagramm. Die Y-Werte können Mittelwert, Median und Geometrischer Mittelwert sein mit optionalen Fehlerbalken aus Standardabweichung, SEM (Standardfehler des Mittelwerts), 95%-KI (oberes 95%-KI des Mittelwerts - unteres 95%-KI des Mittelwerts) oder Bereich (Max-Min).

Normalisieren

Skalieren Sie die Daten neu auf eine allgemeine relative Skala, 0 % bis 100 % oder 0,0 bis 1,0, um gruppen- bzw. teilspaltenübergreifende Vergleiche anzustellen.

Teilspalten Legen Sie fest, wie die Teilspalten in den Gruppen beim Normalisieren behandelt werden sollen.
  • Teilspalten nach Gruppe normalisieren: Alle Teilspalten innerhalb der Gruppe werden zusammen mit Hilfe der Statistik auf Gruppenebene (Zeilenmittelwerte) normalisiert.
  • Jede Teilsalte separat normalisieren: Jede Teilspalte wird einzeln mit Hilfe ihrer eigenen Werte normalisiert.
0 % ist definiert als Legen Sie fest, wie 0 % definiert ist:
  • Falls Teilspalten nach Gruppe normalisieren gilt, definieren Sie 0 % als Kleinster Zeilenmittelwert, Erster Zeilenmittelwert (oder letzter, je nachdem welcher kleiner ist) oder Fester Wert, festgelegt im Bearbeitungsfeld 0 % festgelegt bei.
  • Falls Jede Teilspalte separat normalisieren gilt, definieren Sie 0 % als Kleinster Wert in jeder Teilspalte, Erster Wert in jeder Teilspalte (oder letzter, je nachdem welcher kleiner ist) oder Fester Wert, festgelegt im Bearbeitungsfeld 0 % festgelegt bei.
100 % ist definiert als Legen Sie fest, wie 100 % definiert ist:
  • Falls Teilspalten nach Gruppe normalisieren gilt, definieren Sie 100 % als Größter Zeilenmittelwert, Letzter Zeilenmittelwert (oder erster, je nachdem welcher größer ist), Fester Wert, festgelegt im Bearbeitungsfeld 0 % festgelegt bei, Summe aller Zeilenmittelwerte oder Durchschnitt aller Zeilenmittelwerte.
  • Falls Jede Teilspalte separat normalisieren gilt, definieren Sie 100 % als Größter Wert in jeder Teilspalte, Letzter Wert in jeder Teilspalte (oder erster, je nachdem welcher größer ist), Fester Wert, festgelegt im Bearbeitungsfeld 100 % festgelegt bei, Summe der Werte in jeder Teilspalte oder Durchschnitt der Werte in jeder Teilspalte.
Ergebnisse zeigen als Legen Sie das Ausgabeformat der normalisierten Daten fest.
  • Brüche: Dezimalwert von 0,0 bis 1,0
  • Prozente: Prozente von 0 % bis 100 %

X transformieren

Transformieren Sie die Konzentrationswerte (X-Daten) für die Dosis-Wirkungs-Analyse. Dies ist typischerweise ein erforderlicher Vorverarbeitungsschritt für die sigmoidale Dosis-Wirkungs-Kurvenanpassung.

X = 0 mit anderem Wert ersetzen Wenn X null Konzentrationen enthält (z. B. Fahrzeugkontrollen), aktivieren Sie dieses Kontrollkästchen, um "0" mit einem festgelegten kleinen Wert zu ersetzen. Vor der Log-Transformation erforderlich, da log(0) undefiniert ist.

Geben Sie den Ersatzwert im Bearbeitungsfeld Wert, um X = 0 zu ersetzen ein. Eine gängige Praxis ist es, 0 mit 1/10 der niedrigsten Konzentration, die nicht Null ist, zu ersetzen.

Einheiten ändern Wandeln Sie die Konzentrationseinheiten durch Multiplikation oder Division um.

Wählen Sie Kein, um die Einheiten nicht umzuwandeln. Wenn Multiplizieren oder Dividieren ausgewählt ist, geben Sie den Faktor im Bearbeitungsfeld Anzuwendende Konstante ein. Um beispielsweise "uM" in "mM“ umzuwandeln, wählen Sie Dividieren und 1000 in Anzuwendende Konstante.

In Logarithmen transformieren Wandeln Sie die linearen X-Werte für eine sigmoidale Kurvenanpassung auf eine Log-Skalierung um.

Wählen Sie den Logarithmus aus der Auswahlliste Basis des Logarithmus: Log10 (Basis 10), ln (Basis e) und Log2 (Basis 2).

Nichtlinearer Fit

Führen Sie eine nichtlineare Kurvenanpassung auf XY-Daten durch. Der Dialog NLFit wird aufgerufen, über den Sie den Anpassungsprozess vollständig steuern können.

Funktion auswählen Auf dieser Seite finden Sie ausführliche Informationen zur Anpassungsfunktion (obwohl die verfügbare Kategorie sich vom Standardmodus unterscheiden kann). Klicken Sie auf die Schaltfläche , um die von Ihnen benötigte Funktion zu suchen.

Um Zugriff auf die gesamte Palette der Anpassungsfunktionen zu haben, wechseln Sie zum Standardmodus (Einstellungen: Bedienoberflächenmodus: Standard) und starten Sie Origin erneut.

Anpassungsmethode Legen Sie fest, ob mit den Rohdaten oder der Zusammenfassungsstatistik angepasst werden soll: Mittelwert + StdAbw. (StdAbw. als Gewichtung verwendet), Mittelwert + SE (SE als Gewichtung verwendet) oder Nur Mittelwert. Falls Zusammenfassungsstatistik ausgewählt ist, wird automatisch ein Arbeitsblatt "Summary Stats" erzeugt, das als Eingabedaten verwendet wird.
Globaler Fit Aktivieren Sie dieses Kontrollkästchen, um die Parameter der Anpassungsfunktion für alle Datensätzen (Gruppen) gemeinsam zu nutzen, sodass für alle Datensätze dieselben Parameterwerte verwendet werden.

Deaktivieren Sie diese Option, um jeden Datensatz unabhängig anzupassen und separate Ergebnisse für jeden Datensatz zu erhalten (auch wenn sie in einem Bericht kompiliert werden).

Lesen Sie auf dieser Seite die Einzelheiten zur globalen Anpassung.

Lineare Anpassung

Führen Sie eine lineare Regression für die XY-Daten durch. Jede Gruppe wird unabhängig angepasst, und alle Anpassungsergebnisse werden in einem Bericht kompiliert. Der Dialog Lineare Anpassung wird aufgerufen, über den Sie den Anpassungsprozess vollständig steuern können.

Anpassungsmethode Legen Sie fest, ob mit den Rohdate oder den zusammengefassten Mittelwerten angepasst werden soll.
  • Rohdaten: Jede Gruppe wird unabhängig angepasst, wobei die Replikationen innerhalb von jeder Gruppe als ein Datensatz zusammengefasst und angepasst werden.
  • Mittelwert: Ein Arbeitsblatt "Summary Stats" (Zusammenfassungsstatistik) wird automatisch erzeugt. Jede Gruppe wird unabhängig angepasst, wobei der Mittelwert der Replikationen innerhalb von jeder Gruppe als Eingabe verwendet wird.
Fester Schnittpunkt mit der Y-Achse Beschränken Sie den Schnittpunkt mit der Y-Achse auf den in Fester Schnittpunkt mit der Y-Achse bei festgelegten Wert.
Test auf fehlende Anpassung Aktivieren Sie diese Option, um die Ergebnisse der "fehlenden Anpassung" für replizierte Daten auszugeben, die verwendet werden, um die Modelleignung zu bewerten. Weitere Informationen finden Sie auf dieser Seite.
XY-Wert suchen Aktivieren, um die Arbeitsblätter "FindXfromY" und/oder "FindYfromX" zu erzeugen: Geben Sie einen unabhängigen Variablenwert ein, um den entsprechenden abhängigen Variablenwert zu erhalten oder umgekehrt. Das 95%-Konfidenzintervall wird auch im Blatt berechnet. Weitere Informationen finden Sie auf dieser Seite.
Fehler zeigen Aktivieren Sie diese Option, um Konfidenzbändern und/oder Prognosebänder zum Diagramm der Anpassungskurve hinzuzufügen. Weitere Informationen finden Sie auf dieser Seite.
Residuenanalyse Aktivieren Sie diese Option, um Residuen zu berechnen und auszugeben. Weitere Informationen finden Sie auf dieser Seite.

In Freiform umwandeln

Die aktuelle Datentabelle wird in die typische Form eines Origin-Arbeitsblatt umgewandelt.
