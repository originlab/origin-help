Datenanforderungen

Wählen Sie eine oder mehrere Y-Spalte(n) mit bis zu 4 Faktorspalten.

Diagramm erstellen

Um ein Multi-Vari-Diagramm zu erstellen:

  1. Wählen Sie mindestens eine Arbeitsblattspalte (oder einen Bereich aus mindestens einer Spalte).
  2. Wählen Zeichnen > Statistisch: Multi-Vari-Diagramm, um den Dialog plot_multivari zu öffnen.

Der Dialog plot_multivari

Daten

Wählen Sie die Spalte mit den Eingabedaten. Sie können nur eine Spalte auswählen.

Faktoren

Wählen Sie mehrere Faktoren in Reihenfolge aus. Sie können mehrere Spalten auswählen, aber die Anzahl der Spalten sollte kleiner als 4 sein. Das Ausklappmenü wird deaktiviert, wenn bereits 4 Spalten ausgewählt sind.

Mittelwert für 1. Faktor verbinden

Legen Sie fest, ob die Mittelwertpunkte für den ersten Faktor verbunden werden sollen. Diese Option ist standardmäßig aktiviert.

Mittelwert für 2. Faktor verbinden

Legen Sie fest, ob die Mittelwertpunkte für den zweiten Faktor verbunden werden sollen. Diese Option ist standardmäßig aktiviert. Diese Option ist deaktiviert, wenn es nur 1 Faktor gibt.

Mittelwert für 3. Faktor verbinden

Legen Sie fest, ob die Mittelwertpunkte für den dritten Faktor verbunden werden sollen. Diese Option ist standardmäßig aktiviert. Diese Option ist deaktiviert, wenn es nur 2 Faktoren gibt.

Einzelne Datenpunkte zeigen

Legen Sie fest, on die einzelnen Punkte gezeigt werden sollen. Durch die Aktivierung wird der Boxtyp für alle Boxdiagramme auf Box+Datenüberschneidung und Boxstil auf Keine Box gesetzt. Dieses Kontrollkästchen ist standardmäßig deaktiviert.

Gesamtmittelwertlinie zeigen

Legen Sie fest, ob der Gesamtmittelwert als Referenzlinie bei Mean(plotdata(*,y)) gezeigt werden soll. Dieses Kontrollkästchen ist standardmäßig deaktiviert.

Vorlage

Per Standard wird die Vorlage „multivari“ verwendet, wobei Kontrollkästchen Auto aktiviert ist. Sie können das Kontrolkästchen Auto deaktivieren, um die von Ihnen vorbereitete Vorlage zu verwenden.

Ausgabedaten/Ausgabediagramme

Sobald Sie auf OK geklickt haben, um das Diagramm zu erstellen, können die Zeichnungsdaten und Ergebnisdiagramme in den festgelegten Blättern ausgegeben werden. Per Standard werden zwei neue Blätter mit den Namen "MultiVariPlotData#" und "MultiVariGraph#“ in der aktuellen Arbeitsmappe erstellt, um die Zeichnungsdaten und Ergebnisdiagramme zu speichern.

Vorlage

multivari.otpu (installiert im Origin-Programmordner)

Hinweis

Ein Multi-Vari-Diagramm zeigt die Mittelwerte auf jeder Faktorstufe für jeden Faktor an. Es handelt sich um ein Multi-Boxdiagramm mit Verbindungslinien der Mittelwertpunkte. Jedes Boxdiagramm hat den Boxstil = Keine Box, den Whiskerbereich = Kein, zeigt das Mittelwertperzentil und verbindet die Mittelwerte.

Beispiel

In diesem Beispiel wählen wir 3 Faktoren: "Engine Information.Driveline", "Identification.Classification" und "Identification.Year“ (mit roter Box markiert). Wählen Sie sie als Faktor im Dialog plot_multivari.

Entstapeln Sie die Quelldatenspalten mit Faktor 1 und Faktor 2, einschließlich dem Spaltenfaktor 3 (mit pinker Box markiert); mit 8 entstapelten Spalten zeichnen Sie die Boxdiagramme nur mit verbundenen Mittelwertpunkten (gelb gefüllte Punkte + schwarze Linie). Legen Sie im Dialog Details Zeichnung den Faktor 2 als Teilgruppierung auf der Registerkarte Gruppe und Faktor 3 als Spalte Horizontales Feld auf der Registerkarte Feld fest.

Berechnen Sie die Mittelwerte für die Teilgruppen (markiert mit einer blauen Box) und verschiedene Felder (markiert mit einer grünen Box) und zeichnen Sie sie als Punkt-Liniendiagramme, um diese Mittelwerte darzustellen.
