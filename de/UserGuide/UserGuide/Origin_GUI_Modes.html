Origins Bedienoberflächenmodi

GUI-Modes

Origin bietet während der Installation 3 Modi für die Bedienoberfläche:

  • Standardmodus: Origins klassische Bedienoberfläche mit vollständigen Menüs, Symbolleisten, Schaltflächen und grafischen Optionen.
  • Statistikmodus: Installiert eine Reihe von Apps vor und aktiviert eine Bedienoberfläche, die auf statistische Analysen und Workflows der Qualitätskontrolle zugeschnitten ist.
  • Life-Science-Modus: Installiert Apps vor und ordnet die Bedienoberfläche auf eine Weise an, dass Workflows von pharmakologischen und biologischen Untersuchungen (z. B. Dosis-Wirkungs-Kurven, ANOVA und Lebensdaueranalyse) optimiert sind.


Daher ...

Wenn Sie ein Menü oder eine Schaltfläche nicht finden können oder Schwierigkeiten haben, Anweisungen in unserer Dokumentation zu folgen:

1. Prüfen Sie, ob Sie die Installation für den Modus Statistik oder Life Science durchgeführt haben.

  • Sie können den Bedienoberflächenmodus über das Menü Einstellunen: Bedienoberflächenmodus prüfen:</to>
Wenn Sie den Statistikmodus installiert haben ... Wenn Sie den Life-Science-Modus installiert haben ...

2. Falls das so ist, können Sie entweder:

In den Standardmodus wechseln

Sie können in den Standardmodus wechseln, indem Sie im Menü Einstellungen: Bedienoberflächenmodus: Standard.

Suchfeld verwenden

Geben Sie den Namen eines Menüs oder einer Schaltfläche im Suchfeld neben dem Hauptmenü ein, um das relevante Menü, die Schaltfläche etc. zu finden.
