Datenanforderungen

  • Arbeitsblatt: Wählen Sie XY-Spalten oder XYZ-Spalten (voll oder Teilbereich) oder eine virtuelle Matrix aus.

oder

  • Matrix: Ein Matrixblatt. Ein Blatt mit mehreren Objekten wird unterstützt.

Diagramm erstellen

Aktivieren Sie das Matrixblatt oder wählen Sie die erforderlichen Daten im Arbeitsblatt aus:

  • Wählen Sie im Menü Zeichnen: Kontur: Polare Heatmap θ(X)r(Y).

Wenn Sie die polare Heatmap aus XY-Daten erstellen, wird der Dialog plot_heatmapxy geöffnet.

Wenn Sie die polare Heatmap aus XYZ-Daten erstellen, wird der Dialog plot_heatmapxyz geöffnet.

Wenn Sie die polare Heatmap aus mehreren Y-Spalten ohne Z erstellen, wird der Dialog plotvm geöffnet.

Vorlage

PolarheatmapMat.otpu

(installiert im Origin-Programmordner)

Notizen

Die polare Heatmap ähnelt der Heatmap, wird aber in ein polares Koordinatensystem gezeichnet.

  • Wenn Sie Zeichnen > Kontur: Polare HeatMap θ(X)r(Y) wählen, werden alle X und Y in der Beschriftung in den X-Funktionsdialogen plot_heatmapxy, plot_heatmapxyz und plotvm in θ und R geändert.
  • Es wird das Erstellen einer polaren Heatmap mit RGB-Werten in der Z-Quellspalte oder -Matrix unterstützt.
