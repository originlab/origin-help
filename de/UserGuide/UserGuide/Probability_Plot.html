Datenanforderungen

Wählen Sie mindestens eine Spalte aus, um ein Wahrscheinlichkeitsdiagramm zu erstellen. Wählen Sie mehrere Daten- und Gruppenspalten, um die Verteilung von verschiedenen Daten/Gruppen zu vergleichen.

Diagramm erstellen

Wählen Sie die erforderlichen Daten aus

Wählen Sie Zeichnen > Statistisch: Wahrscheinlichkeitsdiagramm...

Oder:

Klicken Sie auf die Schaltfläche Wahrscheinlichkeitsdiagramm auf der Symbolleiste 2D Grafiken.

Origin öffnet den Dialog plot_prob.

Passen Sie die Optionen im Dialog benutzerdefiniert an und klicken Sie dann auf OK, um das Diagramm zu erstellen. Bitte lesen Sie weitere Einzelheiten zum Erstellen und benutzerdefinierten Anpassen von Wahrscheinlichkeitsdiagrammen.

Vorlage

Abhängig von dem ausgewählten Verteilungstyp wird eine der folgenden Vorlagen verwendet:

  • gPPNormal.optu
  • gPPLog.optu
  • gPPExp.optu
  • gPPWeibull.optu
  • gPPGamma.optu

(installiert im Origin-Programmordner)
