Zeilen reduzieren

reduce-rows


Origin verfügt über ein Hilfsmittel zum Reduzieren von Daten durch Löschen bzw. Zusammenfügen von Arbeitsblattzeilen gemäß einem benutzerdefinierten Muster oder durch Löschen von maskierten Zeilen.

Zum Öffnen des Hilfsmittels:

  1. Aktivieren Sie die Arbeitsmappe.
  2. Klicken Sie auf Arbeitsblatt: Zeilen reduzieren, und der Dialog wreducerows wird geöffnet.

Der Dialog wreducerows beruht auf der X-Funktion wreducrows.

Dialogoptionen

Eingabe

Der Dateneingabebereich

Hilfe zum Festlegen von Bereichen finden Sie hier: Eingabedaten festlegen
Reduktionsmethoden
  • N Zeilen löschen, dann M Zeilen überspringen
N Zeilen werden gelöscht, dann werden M Zeilen übersprungen.
  • Zeilen mit fehlenden Werten löschen
Es werden die Zeilen mit fehlenden Werten gelöscht.
  • Zeilen mit maskierten Werten löschen
Es werden die Arbeitsblattzeilen gelöscht, die maskiert wurden.
  • N Zeilen mit zusammengefügten Werten reduzieren
Alle N Datenpunkte werden in einem Datenpunkt zusammengefügt entsprechend dem Wert Zusammenfügen mit.
  • Verborgene Zeilen löschen
Verborgene Zeilen löschen (z. B. Daten herausfiltern)
Ganze Zeile löschen

Diese Option ist nur verfügbar, wenn unter Reduktionsmethode die Option Zeilen mit fehlenden Werten löschen oder Zeilen mit maskierten Werten löschen gewählt ist. Wählen Sie in der Auswahlliste, um die Zeilen zu löschen, in denen fehlende oder maskierte Werte vorliegen, oder löschen Sie nur die fehlenden oder maskierten Zellen.

  • Zeilen in Auswahl
Reduzieren Sie Zeilen, indem Sie die Teilzeilen in den Eingabespalten basierend auf fehlenden/maskierten Werten löschen.
  • Ganze Zeile des Blatts
Reduzieren Sie die gesamte Zeile des Eingabearbeitsblatts basierend auf fehlenden/maskierten Werten.
  • Nur Zellen
Nur gelöschte fehlende Werte oder maskierte Zellen.
Zeilen hinzufügen/löschen

Die Option hängt von Ihrer Auswahl der Reduktionsmethode: N Zeilen löschen... (Zeilen löschen) oder N Zeilen reduzieren... (Zeilen zusammenfügen) ab. Anzahl der zusammenzufügenden oder zu löschenden Zeilen
Zeilen überspringen

Option ist nur verfügbar, wenn Reduktionsmethode = N Zeilen löschen, dann M Zeilen überspringen. Legen Sie die Anzahl der zu überspringenden Zeilen fest.
Zusammenfassen mit

Option ist nur verfügbar, wenn Reduktionsmethode = N Zeilen mit zusammengefügten Werten reduzieren. Legen Sie den Wert fest, der zusammengefügte Datenpunkte ersetzen soll.

  • Erster Punkt
Alle N Datenpunkte werden mit dem Wert des ersten zusammengefügten Punkts ersetzt.
  • Letzter Punkt
Alle N Datenpunkte werden mit dem Wert des letzten zusammengefügten Punkts ersetzt.
  • Durchschnitt
Alle N Datenpunkte werden mit dem Wert des Durchschnitts der zusammengefügten Punkte ersetzt.
  • Min.
Alle N Datenpunkte werden mit dem Wert des Minimums der zusammengefügten Punkte ersetzt.
  • Max.
Alle N Datenpunkte werden mit dem Wert des Maximums der zusammengefügten Punkte ersetzt.
  • Summe
Alle N Datenpunkte werden mit dem Wert der Summe der zusammengefügten Punkte ersetzt.
  • StdAbw.
Alle N Datenpunkte werden mit der Standardabweichung der zusammengefügten Punkte ersetzt.
Erste zusammenzufügende/zu löschende Zeile

Die erste Zeile, die zusammengefügt bzw. gelöscht werden soll
Ausgabe

Legen Sie den Ausgabebereich fest.
