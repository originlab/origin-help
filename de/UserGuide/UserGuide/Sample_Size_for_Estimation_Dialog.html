Parameter

Wählen Sie eine der folgenden Statistikwerte als Parameter:

  • Mittelwert (Normal)
  • Standardabweichung (Normal)
  • Varianz (Normal)
  • Anteil (Binomial)
  • Rate (Poisson)
  • Mittelwert (Poisson)

Parameter sind deskriptive Maße einer ganzen Grundgesamtheit, die als Eingaben verwendet werden.

Parameterwert

Nach Auswahl des Parameters in der obenstehenden Liste geben Sie den Wert des ausgewählten Parameters ein.

Parameter Wert
Mittelwert (Normal) Standardabweichung
Standardabweichung (Normal) Standardabweichung
Varianz (Normal) Varianz
Anteil (Binomial) Anteil
Rate (Poisson) Rate
Mittelwert (Poisson) Mittelwert

Berechnen

Stichprobenumfänge Berechnen Sie den Stichprobenumfang für die gewünschte(n) Fehlerspanne(n).
Fehlerspannen Berechnen Sie die Fehlerspannen für die gegebenen Stichprobenumfänge.

Fehlerspannen

Wenn Stichprobenumfänge in der Auswahlliste Berechnen ausgewählt ist, wird das Kästchen Fehlerspannen in diesem Dialog gezeigt.

Sie können einen Wer für die gewünschte Fehlerspanne festlegen. Durch Eingabe mehrerer Werte lässt sich der Effekt der unterschiedlichen Fehler am Rand der Spanne auswerten. Normalerweise sind die Stichprobenumfänge größer, wenn die eingegebenen Fehlerspannen kleiner sind.

Stichprobenumfänge

Wenn Fehlerspannen in der Auswahlliste Berechnen ausgewählt ist, wird das Kästchen Stichprobenumfänge in diesem Dialog gezeigt.

Sie können einen Wert für den Stichprobenumfang festlegen. Durch Eingabe mehrerer Werte kann der Effekt von verschiedenen Stichprobenumfängen ausgewertet werden. Normalerweise werden die Fehlerspannen kleiner, wenn die eingegebenen Stichprobenumfänge größer werden.

Konfidenzniveau (%)

Legen Sie das Konfidenzniveau für das Konfidenzintervall fest. Typischerweise wird das Konfidenzniveau mit 95 % festgelegt. Abhängig vom Stichprobenumfang müssen Sie jedoch in Betracht ziehen, das Konfidenzniveau auf einen anderen Wert zu ändern. Wenn der Stichprobenumfang groß ist, ist ein hohes Konfidenzniveau erforderlich.

Konfidenzintervall

Zweiseitig Verwenden Sie den zweiseitigen Test, um zu bestimmen, ob der Parameter der Grundgesamtheit sich vom hypothetischen Wert unterscheidet.
Untere Schranke Verwenden Sie den einseitigen Test, um zu bestimmen, ob der Parameter der Grundgesamtheit kleiner ist als der hypothetische Wert.
Obere Schranke Verwenden Sie den einseitigen Test, um zu bestimmen, ob der Parameter der Grundgesamtheit größer ist als der hypothetische Wert.

Annehmen, dass die Standardabweichung der Grundgesamtheit bekannt ist

Diese Option ist nur verfügbar, wenn der Parameter der Mittelwert (Normal) ist. Wenn der Wert der Standardabweichung der Grundgesamtheit bekannt und die Hoffnung ist, den Stichprobenumfang oder die Fehlerspanne für Z bei einer Stichprobe zu schätzen, können Sie diese Option auswählen.

Exposition

Das Kontrollkästchen Exposition ist nur verfügbar, wenn der Parameter die Rate (Poisson) ist. Geben Sie einen Wert ein, um die Stichprobenrate des Vorkommens in eine nützlichere Form aufzuteilen.

Ergebnisse ausgeben

Bestimmen Sie, wo das auszugebende Berichtsblatt erstellt wird. Weitere Einzelheiten finden Sie unter: Analyseberichtsblätter.
