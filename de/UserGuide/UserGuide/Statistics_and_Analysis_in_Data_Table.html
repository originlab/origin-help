Statistik in Datentabelle

Für folgende häufig verwendete Werkzeuge von gruppierten und XY-Tabellen bietet der Life-Science-Modus einen Schnellstartdialog und wählt automatisch die geeigneten Eingabedaten aus, um den Workflow der Analyse zu vereinfachen. Der Dialog mit sämtlichen Funktionen wird dann geöffnet und gewährt Ihnen die vollständige Kontrolle, während Ihr Workflow optimiert wird.

t-Test bei einer Stichprobe

  1. Wählen Sie im Menü Statistik: Hypothesentests: t-Test bei einer Stichprobe.
  2. Wählen Sie die Eingabedatenanordnung im geöffneten Dialog:
    Replikationen vor Analyse mitteln Die Werte in jeder Zeile werden gemittelt. Dann wird die Analyse auf die gemittelten Werte durchgeführt.
    Jede Teilspalte separat analysieren Die Analyse wird unabhängig für jede Teilspalte durchgeführt.
    Teilspalten vor Analyse zusammenfassen Alle Teilspaltenwerte stapeln und sie dann als eine Stichprobe analysieren

Lesen Sie auf dieser Seite die Informationen zu allen Bedienelementen des t-Test bei einer Stichprobe.

Zweifache ANOVA

  1. Wählen Sie im Menü Statistik: ANOVA: Zweifache ANOVA.
  2. Vergeben Sie im aufgerufenen Dialog den zeilenweisen und spaltenweisen Faktoren jeweils einen Namen.

Lesen Sie auf dieser Seite die Informationen zu allen Bedienelementen der zweifachen ANOVA.

Zweifache ANOVA mit wiederholten Messungen

  1. Wählen Sie im Menü Statistik: ANOVA: Zweifache ANOVA bei wiederholten Messungen.
  2. Im geöffneten Dialog:
    Subjekte definieren Legen Sie fest, wie Subjekte definiert werden. Die Anzahl der Subjekte wird zur Bestätigung in Klammern angezeigt.
    Faktornamen Vergeben Sie den zeilenweisen und spaltenweisen Faktoren jeweils einen Namen.


    Lesen Sie auf dieser Seite die Informationen zu allen Bedienelementen der zweifachen ANOVA bei wiederholten Messungen.
