Stats-Mode

Was ist der Statistikmodus?

Der Statistikmodus wurde mit Origin 2025b eingeführt, um die Statistikroutine mehr in den Fokus zu stellen. Dieser Modus bereinigt die Origin-Bedienoberfläche, indem ein Großteil der Grafikschaltflächen und der erweiterten Analysewerkzeuge verborgen werden und das Menü Grafik neu strukturiert wird, um den Zugriff auf statistische Diagramme zu erleichtern und einfacher navigieren zu können. Außerdem werden beliebte Statistik-Apps, wie DOE, vorinstalliert.


Da der Statistikmodus nur einige einfache Menüs und Schaltflächen behält, nutzen Sie bitte das Suchfeld, wenn Sie ein Hilfsmittel nicht auf der Bedienoberfläche finden können, um es zu finden und auszuführen.

Wie wird der Statistikmodus aktiviert?

  • Statistikmodus während der Installation aktivieren
    Beim Installieren von Origin haben Sie Optionen, um eine Sammlung von Origins statistischen Apps vorzuinstallieren und den Statistikmodus auf der Seite "Modus der grafischen Bedienoberfläche (GUI) auswählen" zu aktivieren.

    Wenn Sie das getan haben, wird Origin per Standard im Statistikmodus gestartet.
    Hinweis: Lesen Sie auf dieser Seite eine Schritt-für-Schritt-Anweisung zur Origin-Installation.
  • Wenn Sie Origin mit der '''Standardbedienoberfläche''‘ installiert haben,
    können Sie immer in den Statistikmodus wechseln, indem Sie “Origin modifizieren”:
    1. Wählen Sie im Menü Origin reparieren oder deinstallieren #### im Startmenü von Windows.
    2. Wählen Sie Modifizieren auf der Willkommensseite.
    3. Klicken Sie Weiter bis zur Seite „Modus der grafischen Bedienoberfläche (GUI) auswählen". Wählen Sie die Option Statistikbedienoberfläche.
    4. Klicken Sie auf Weiter, um die Installation abzuschließen.
    Hinweise:

    Wenn Sie das Kontrollkästchen Statistikbedienoberfläche aktivieren, ist eine der vorinstallierten Apps Design of Experiments (DOE). Diese App erfordert, dass die R-Software in der Version 4.4.1 VOR OriginPro installiert wurde. Bitte verwenden Sie den untenstehenden Link, um diese spezielle Version der Software R zu erhalten:

    • Index of /bin/windows/base/old/4.4.1

Wie wird der Statistikmodus ausgeschaltet?

Sie können immer zur klassischen Bedienoberfläche zurückkehren:

  • Wählen Sie im Menü Einstellungen: Modus: Standard.
