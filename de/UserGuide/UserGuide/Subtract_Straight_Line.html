Gerade Linie subtrahieren

Subtract-Straight-Line

Beschreibung

Dieses Hilfsmittel kann verwendet werden, um eine gerade Linie von einem Datensatz zu subtrahieren. Die gerade Linie wird von zwei Punkten bestimmt. Wenn ein Diagrammfenster aktiv ist, können Sie dieses Hilfsmittel im Menü verwenden.

Um eine gerade Linie zu subtrahieren:
  • Wenn ein Diagrammfenster aktiv ist, wählen Sie Analyse: Mathematik: Gerade Linie subtrahieren.

oder

  • Wenn ein Arbeitsblatt aktiv ist, führen Sie subtract_line -d; im Skriptfenster aus, um den Dialog zu öffnen.

Dialogoptionen

Neuberechnen

Bedienelemente zur Neuberechnung der Analyseergebnisse

  • Kein
  • Auto
  • Manuell

Weitere Informationen finden Sie unter Analyseergebnisse neu berechnen.

Eingabe

Legt den Bereich der XY-Daten oder der Kurve fest, die subtrahiert werden.

Hilfe zum Festlegen von Bereichen finden Sie hier: Eingabedaten festlegen

Linie definieren mit

Wählen Sie die Methode zum Definieren der geraden Linie aus:

  • Zwei Punkte:
    Legt die X- und Y-Werte der Anfangs- und Endpunkte für die gerade Linie fest.
    Diese zwei Punkte können auch im Diagramm über die Schaltfläche Punkte auswählen ausgewählt werden. Klicken Sie doppelt auf das Diagramm, um die Punkte auszuwählen.
  • Linie aus ausgewähltem Bereich anpassen:
    Legt den Linienbereich aus einer Zeichnung in der Grafik oder einer Spalte im Arbeitsblatt fest.
Ausgabe

Legt den Datenbereich der Ausgabe fest. Die Ausgabekurve wird in die Grafik eingefügt.

Hilfe zum Festlegen der Bereiche finden Sie unter: Ergebnisse ausgeben
