Die Registerkarte Beschriftungen (Details Zeichnung) für Ursache-Wirkung

PD-Labels-tab-CauseEffect

Diese Registerkarte wird verwendet, um die Beschriftungseinstellungen für das Ursache-Wirkungsdiagramm vorzunehmen.

Wirkung

Sie können die Bedienelemente verwenden, um die Schriftart festzulegen, einen Rahmen hinzuzufügen oder den Text für die Beschriftung der Wirkung umzubrechen.

Hauptursachen

Sie können die Bedienelemente verwenden, um die Schriftart festzulegen, einen Rahmen hinzuzufügen oder den Text für die Beschriftungen der Hauptursachen umzubrechen.

Sub-Ursachen

Sie können die Bedienelemente verwenden, um die Schriftart festzulegen, einen Rahmen hinzuzufügen oder den Text für die Beschriftungen der Sub-Ursachen umzubrechen.
