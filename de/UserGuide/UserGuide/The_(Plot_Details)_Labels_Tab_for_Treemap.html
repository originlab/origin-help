Schriftart

Legen Sie die Schriftart, die Schriftgröße und andere Stile für alle Beschriftungen fest.

Position

Legen Sie die Position der Beschriftungen auf den Rechtecken fest.

Format

Legen Sie den Beschriftungsinhalt und das Anzeigeformat für die Beschriftungen fest.

Überschreitung

Legen Sie fest, wie mit Beschriftungen umgegangen werden soll, die zu lang sind.

Beschriftung verbergen, wenn Prozent weniger als (%)

Verbergen Sie alle Beschriftungen der Rechtecke, wenn der Prozentwert (in der Gesamtgewichtung aller Y-Werte) kleiner ist als ein angegebener Wert.

Gruppenbeschriftungen

Zeigen

Legen Sie die Gruppenbeschriftungen der Ebenen fest, die gezeigt werden.

Platzierung

Legen Sie fest, wo die Gruppenbeschriftungen platziert werden sollen.

Position

Legen Sie die Position der Gruppenbeschriftungen fest.

Schriftart

Die gleichen Schriftbedienelemente wie für die Beschriftungen auf den Rechtecken (die Beschriftungen für letzte Ebene). Per Standard werden automatisch die Einstellungen der obigen Schriftsbedienelemente befolgt.

Hintergrund

Legen Sie die Hintergrundfarbe für die Gruppenbeschriftungen fest. Sie können eine Inkrementfarbliste auswählen, um verschiedene inkrementelle Farben auf verschiedene Gruppen/Teilgruppen anzuwenden.

Rahmen

Wenn Platzierung = Überlagerung, können Sie eienn Rahmen zur Gruppenbeschriftung hinzufügen.

Überschreitung

Legen Sie fest, wie mit der Überschreitung der Beschriftungen umgegangen werden soll. Die Option ist die gleiche wie das Bedienelement Überschreitung oben.
