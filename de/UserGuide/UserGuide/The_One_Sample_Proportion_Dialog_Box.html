Eingabe

Form der Eingabedaten
  • Zusammengefasst
Führen Sie den Test mit Hilfe einer zusammenfassenden Statistik durch, die Sie manuell in den Dialog eingeben.
  • Roh
Führen Sie den Test mit Hilfe eines Datensatzes durch, der aus "Erfolgs-" bzw. "Fehler"werten besteht. Der Datensatz darf nicht mehr als zwei eindeutige Werte enthalten.

Test von Anteilen

Testanteil Der zu testende hypothetische Anteil
Nullhypothese Die zu testende Nullhypothese (Stichprobenanteil = Testanteil).
Alternativhypothese Optionen:
Proportion <> Testanteil
Ein beidseitiger Test von Anteilen, der zu beantworten versucht, ob der Anteil der Stichprobe = Testanteil.
Proportion > Testanteil
Ein einseitiger Test von Anteilen nach oben, der zu beantworten versucht, ob der Anteil der Stichprobe größer ist als der Testanteil.
Proportion < Testanteil
Ein einseitiger Test von Anteilen nach unten, der zu beantworten versucht, ob der Anteil der Stichprobe kleiner ist als der hypothetische Anteil.
Signifikanzniveau Das Signifikanzniveau des Tests
Kondenzintervall(e) Berechnen Sie das Konfidenzintervall für den wahren (Grundgesamtheit) Anteil.
Konfidenzniveau(s) in % Legen Sie die Konfidenzniveaus fest, für die die Konfidenzintervalle berechnet werden. Diese Option ist nur verfügbar, wenn Konfidenzintervall(e) aktiviert ist.
Testmethode Origin verwendet zwei Methoden zum Berechnen von p-Werten und Konfidenzintervallen für Tests von Anteilen bei einer Stichprobe (Dokumentation zur Approximation der Normalverteilung und zum Binomialtest finden Sie auf der Seite zum Algorithmus):
  • Approximation der Normalverteilung
Die Approximation der Normalverteilung wird immer berechnet. Im Allgemeinen erhalten Sie ein befriedigendes Ergebnis, wenn Erfolge ≥ 10 und wenn Stichprobenumfang - Erfolge ≥ 10. Die Qualität der Approximation neigt dazu, sich mit zunehmendem Stichprobenumfang zu verbessern.
  • Binomialtest
Der Binomialtest wird verwendet.

Testmethode

Normal-Approximation Die Approximation der Normalverteilung wird immer berechnet. Im Allgemeinen erhalten Sie ein befriedigendes Ergebnis, wenn Erfolge ≥ 10 und wenn Stichprobenumfang - Erfolge ≥ 10. Die Qualität der Approximation neigt dazu, sich mit zunehmendem Stichprobenumfang zu verbessern.
Binomialtest Der Binomialtest wird verwendet.

Analyse der Trennschärfe

Mindestdifferenz von Interesse Legen Sie die Mindestdifferenz vom Testmittelwert fest, für den Sie sich interessieren. Sobald Sie einen Wert in dieses Feld eingeben, zeigt das Diagramm der Trennschärfeanalyse den Mindeststichprobenumfang, um diese Differenz in verschiedenen Trennschärfen zu entdecken.
Tatsächliche Trennschärfe Berechnen Sie die Trennschärfe eines Tests (die Wahrscheinlichkeit, die Nullhypothese abzulehnen, wenn sie falsch ist).
Signifikanzniveau Diese Option ist nur verfügbar, wenn Hypothetische Trennschärfe und/oder Tatsächliche Trennschärfe aktiviert ist/sind. Dies ist das Signifikanzniveau (Alpha-Wert) für die Berechnung der Trennschärfe.
Hypothetische Trennschärfe Berechnen Sie die Trennschärfe des Tests für verschiedene Stichprobenumfänge.
Hypothetische/r Stichprobenumfang/-umfänge Nur verfügbar, wenn Hypothetische Trennschärfe aktiviert ist. Legen Sie den/die hypothetischen Stichprobenumfang/-umfänge für die Berechnung der Trennschärfe fest.

Ausgabe

Hilfe zum Festlegen der Bereiche finden Sie unter: Ergebnisse ausgeben

Ausgabe Ergebnisprotokoll Legen Sie die Ergebnisse fest, die im Ergebnisfenster ausgegeben werden.
Bericht der grafischen Zusammenfassung Legen Sie fest, ob der grafische Zusammenfassungsbericht angezeigt werden soll.
Grafischer Bericht der Trennschärfeanalyse Legen Sie fest, ob der grafische Bericht der Trennschärfeanalyse angezeigt werden soll. Die Trennschärfekurve zeigt, wie die statistische Trennschärfe des Tests sich ändert, wenn die Differenz zwischen den Anteilen wechselt. Die Trennschärfe ist die Wahrscheinlichkeit, dass die Nullhypothese verworfen wird, wenn die wahre Differenz sich von 0 unterscheidet.
Ergebnisse ausgeben Legen Sie das Ausgabearbeitsblatt für die Testergebnisse fest.
Zeichnungsdaten ausgeben Legen Sie das Arbeitsblatt für die Ausgabe der Zeichnungsdatensätze fest.
