Führen Sie den Test auf die Eingabedaten durch, die aus einer Stichprobendatenspalte und einer „Gruppierung“sspalte bestehen. Die Stichprobendatenspalte sollte in Form von "Erfolg" oder "Fehler" vorliegen und darf nicht mehr als zwei eindeutige Werte enthalten. Die Gruppierungsspalte sollte Informationen für die zwei Grundgesamtheiten von Interesse enthalten, indexiert (nach Zeilennummer) auf die Spalte der Stichprobendaten.

Test von Anteilen

Testdifferenz Die zu testende hypothetische Anteilsdifferenz (allgemein "0").
Nullhypothese Die zu testende Nullhypothese
Alternativhypothese Optionen:
Anteil 1 - Anteil 2<> Testdifferenz
Führen Sie einen zweiseitigen Test zur Hypothese durch, dass Anteil1 - Anteil2 = Testdifferenz.
Anteil 1 - Anteil 2 > Testdifferenz
Führen Sie einen einseitigen Test nach oben zur Hypothese durch, dass Anteil1 - Anteil2 größer ist als Testdifferenz.
Anteil 1 - Anteil 2 < Testdifferenz
Führen Sie einen einseitigen Test nach unten zur Hypothese durch, dass Anteil1 - Anteil2 kleiner ist als Testdifferenz.
Signifikanzniveau Das Signifikanzniveau des Tests
Kondenzintervall(e) Berechnen Sie die Konfidenzintervalle für die Anteildifferenzen.
Konfidenzniveau(s) in % Legen Sie die Konfidenzniveaus fest, für die die Konfidenzintervalle berechnet werden. Diese Option ist nur verfügbar, wenn Konfidenzintervall(e) aktiviert ist.
Testmethode Um mehr über die Testmethoden zu erfahren, siehe die Seite zu den Algorithmen.
  • Normal-Approximation
Berechnen Sie den p-Wert und das Konfidenzintervall im Ergebnis der Approximation der Normalverteilung.
  • Gepooltes p für Test verwenden
Legen Sie fest, eine gepoolte Schätzung von p für den Test zu verwenden, wenn die Testdifferenz 0 ist.
  • Fishers exakter Test
Berechnen Sie den p-Wert und das Kondenzintervall mit Fishers exaktem Test, der auf dem Binomialtest beruht.

Trennschärfeanalyse

Kontrollkästchen Trennschärfeanalyse: Bei Aktivierung wird die Trennschärfe für den Test von Anteilen berechnet.

Tatsächliche Trennschärfe Ist dieses Kontrollkästchen aktiviert, wird die Ist-Trennschärfe berechnet.
Signifikanzniveau Legen Sie das Signifikanzniveau () für die Berechnung der Trennschärfe fest. Diese Option ist nur verfügbar, wenn entweder die tatsächliche Trennschärfe oder die Hypothetische Trennschärfe aktiviert ist.
Hypothetische Trennschärfe Ist dieses Kontrollkästchen aktiviert, wird die hypothetische Trennschärfe berechnet.
Hypothetische/r Stichprobenumfang/-umfänge Legen Sie den/die hypothetischen Stichprobenumfang/-umfänge fest, für den/die die Trennschärfe berechnet wird. Diese Option ist nur verfügbar, wenn Hypothetische Trennschärfe aktiviert ist.

Ausgabe

Hilfe zum Festlegen der Bereiche finden Sie unter: Ergebnisse ausgeben

Ausgabe Ergebnisprotokoll Legen Sie die Ergebnisse fest, die im Ergebnisfenster ausgegeben werden.
Bericht der grafischen Zusammenfassung Legen Sie fest, ob der grafische Zusammenfassungsbericht angezeigt werden soll.
Grafischer Bericht der Trennschärfeanalyse Legen Sie fest, ob der grafische Bericht der Trennschärfeanalyse angezeigt werden soll. Die Trennschärfekurve zeigt, wie die statistische Trennschärfe des Tests sich ändert, wenn die Differenz zwischen den Anteilen wechselt. Die Trennschärfe ist die Wahrscheinlichkeit, dass die Nullhypothese verworfen wird, wenn die wahre Differenz sich von 0 unterscheidet.
Ergebnisse ausgeben Legen Sie das Ausgabearbeitsblatt für die Testergebnisse fest.
Zeichnungsdaten ausgeben Legen Sie das Arbeitsblatt für die Ausgabe der Zeichnungsdatensätze fest.
