Der Dialog legendupdate

 

legendupdate-dlg

Der Dialog legendupdate wird verwendet, um Legenden auf der Diagrammseite/im Layer zu aktualisieren oder zu rekonstruieren. Der Zugriff auf ihn erfolgt bei aktivem Diagrammfenster durch Auswahl von Grafik: Legende: Legende aktualisieren. Dieser Dialog verwendet die X-Funktion legendupdate.

Die folgende Tabelle enthält Referenzen für die Bedienelemente in diesem Dialog:

Umfang aktualisieren

Legt fest, wo die Aktualisierung der Legende angewendet werden soll.

  • Gesamte Seite
    Aktualisiert die Legenden auf der gesamten aktiven Seite.
  • Aktiver Layer
    Aktualisiert die Legenden auf der aktiven Seite.
Modus Aktualisieren

Legt fest, ob die Legende rekonstruiert oder aktualisiert werden soll.

  • Aktualisieren
    Aktualisiert die Daten der vorhandenen Legende(n) über Automatischer Übersetzungsmodus der Legende. Das Format und die Struktur der vorhandenen Legende(n) werden nicht verändert.
  • Rekonstruieren
    Die Legende(n) wird entsprechend der Einstellungen für Automatischer Übersetzungsmodus der Legende und Ordnung neu erstellt. Das Legendenformat wird auf die Standardeinstellungen zurückgesetzt.
Legende

Diese Option ist nur dann verfügbar, wenn der Aktualisierungsmodus auf Rekonstruieren gesetzt wurde. Dies wird verwendet, um den Legendentyp festzulegen, den Sie rekonstruieren möchten.

  • Legende for jedes Diagramm
    Erzeugt eine separate Legende für jeden Layer.
  • Eine Legende für gesamte Seite
    Erzeugt eine Legende für alle Layer im Diagramm.
Ordnung

Diese Option ist nur dann verfügbar, wenn der Aktualisierungsmodus auf Rekonstruieren gesetzt wurde. Hiermit wird die Zeichnungsordnung in der rekonstruierten Legende festgelegt. Sie kann entweder Aufsteigend oder Absteigend sein. Beachten Sie, dass Sie für eine Legende, die nie manuell aktualisiert wurde, auch Grafik: Legende: Umgekehrte Ordnung verwenden können, um die Einträge der Datenzeichnungen umzukehren, ohne die Legende rekonstruieren zu müssen.
Zeichnungsreihenfolge innerhalb des Layers

Dies ist nur verfügbar, wenn der Aktualisierungsmodus auf Rekonstruieren und Legende auf Eine Legende für gesamte Seite gesetzt ist. Diese Option steuert die Zeichnungsreihenfolge, die innerhalb eines Layers abgelesen wird.

  • Absteigend
    Die Zeichnungsinformationen innerhalb eines Layers werden gemäß der Indizes in absteigender Reihenfolge abgelesen.
  • Aufsteigend
    Die Zeichnungsinformationen innerhalb eines Layers werden gemäß der Indizes in aufsteigender Reihenfolge abgelesen.
Automatischer Übersetzungsmodus der Legende

Zum Erstellen der Legenden werden die Notationen der LabTalk-Variablen ("%(1)", "%(2)" etc.) verwendet. Origin "übersetzt" beim Erstellen einer Legende die Notationen automatisch in Informationen. Diese Variable legt die Informationen fest, die verwendet werden, um die Notationen zu übersetzen.

Wenn die Option Benutzerdefiniert gewählt ist, können Sie eine benutzerdefinierte Notation verwenden. Ansonsten können Sie den Datenbereichsnamen oder eine Variable in den Spaltenbeschriftungszeilen verwenden. Einzelheiten zu den unterstützten Variablennotationen erhalten Sie in der Tabelle der Textbeschriftungsoptionen.

Wenn die volle Syntax gewählt ist, können die Bearbeitungsfelder der vollen Syntax verwenden, um das Legendenformat benutzerdefiniert anzupassen.
Benutzerdefiniertes Legendenformat (@D,@LU etc.) Diese Option ist nur verfügbar, wenn Benutzerdefiniert unter Automatischer Übersetzungsmodus der Legende gewählt wurde. Es wird ein Auswahlmenü angezeigt, das mehrere häufig verwendete Notationskombinationen enthält. Die Kombination der Legendennotationen mit einem beliebigen Buchstaben vor, nach und dazwischen ist erlaubt. Beispiel:
@LL\+(o)C: Spaltenlangname der Datenzeichnung plus Grad Celsius.

Sie können auf die Schaltfläche neben dem Kombinationsfeld klicken, um die Liste der @Options anzuzeigen. Sie können in der Tabelle der Textbeschriftungsoptionen die unterstützten Notationen nachlesen.

Um den Legendentext umzubrechen, setzen Sie den Automatischen Übersetzungsmodus der Legende auf Benutzerdefiniert und fügen Sie "%(CRLF)" in Ihren Legendentext bei dem Punkt ein, wo Sie den Text in eine neue Zeile umbrechen möchten.
Zusätzliches Format (für @L, @V etc.) Dies ist verfügbar, um erforderliche zusätzliche Parameter hinzuzufügen, wenn @L, @V oder @WT unter Benutzerdefiniertes Legendenformat (@D, @LU etc) ausgewählt ist. @L = X-Wert. @V = Y-Wert, @WT = Zellenwert. Zum Beispiel: 
Um den 1. X-Wert als Legendentext zu zeigen, geben Sie @L in Benutzerdefiniertes Legendenformat und 1 im Feld Zusätzliches Format (für @L, @V etc.) ein. Um den 2. Y-Wert als Legendentext zu zeigen, geben Sie @V in Benutzerdefiniertes Legendenformat und 2 im Feld Zusätzliches Format (für @L, @V etc.) ein. Um den Zellenwert in Spalte C, Zeile 4 zu zeigen, geben Sie @WT in Benutzerdefiniertes Legendenformat und 3, 4 oder C, 4 im Feld Zusätzliches Format (für @L, @V etc.) ein.

Einzelheiten erhalten Sie in der Tabelle der Textbeschriftungsoptionen.
Volle Syntax Wenn Volle Syntax in der Auswahlliste Autom. Legend ausgewählt ist, können Sie mit der vollen Syntax des Legendentexts füllen.

Es gibt ein Ausklappmenü, um Beispiele der Syntax zu zeigen.

Zum Beispiel bedeutet %(?X, @LL) Langname der X-Spalte. ? stellt den Zeichnungsindex dar.
Legende nur für sichtbare Zeichnungen zeigen

Dieses Kontrollkästchen legt fest, ob die verborgene Zeichnung in der Legende gezeigt wird oder nicht. Wenn das Kontrollkästchen aktiviert wird, wird die verborgene Zeichnung nicht in der Legende gezeigt.
Hinweis: Sie können auch mit der rechten Maustaste auf die Legende klicken und Legende: Legende nur für sichtbare Zeichnungen zeigen wählen können, um die verborgenen Zeichnungen zu verbergen.

Wenn dieses Bedienelement aktiviert ist und Sie eine Zeichnung verbergen/entfernen, wird das Legendenfeld neu erzeugt und aktualisiert, um nur die verbleibenden Zeichnungen zu zeigen.
Legende für Anpassungskurven verbergen

Dieses Kontrollkästchen legt fest, ob die angepasste Kurve in der Legende verborgen wird oder nicht. Wenn das Kontrollkästchen aktiviert wird, wird die Anpassungskurve nicht in der Legende gezeigt. Beachten Sie, dass Sie auch mit der rechten Maustaste auf die Legende klicken und Legende: Legende für angepasste Kurven verbergen wählen können, um die Kennzeichen für den aktiven Datensatz zu verbergen.
Aktiven Datensatz kennzeichnen Dieses Kontrollkästchen bestimmt, ob der aktive Datensatz in der Legende gekennzeichnet wird. Wenn es aktiviert ist, wird das Zeichnungssymbol des aktiven Datensatzes in der Legende mit einem roten Rechteck umrahmt.

Beachten Sie, dass Sie beide der folgenden Methoden wählen können, um die Kennzeichnung des aktiven Datensatzes anzuzeigen:

  • Klicken Sie mit der rechten Maustaste auf die Legende und wählen Sie Legende: Aktiven Datensatz kennzeichnen.
  • Wählen Sie Format: Seite und gehen Sie zur Registerkarte Legenden/Titel. Aktivieren Sie unten Aktiven Datensatz kennzeichnen.
