Die Bedienelemente des allgemeinen Dialogfelds

Designs werden über Registerkarten organisiert: Die Standarddesigns befinden sich im Pfad = System. Standardmäßig werden benutzerdefinierte Design in Ihrem Anwenderdateiordner gespeichert.

Name

Der Name einer Designdatei
Größe

Die Größe einer Designdatei
Datum

Das Erstellungsdatum eines Designs
Pfad

Der Speicherort der Designdatei, die entweder für System (für mit Origin ausgelieferte Designs) oder für Anwender (für von Anwendern erstellte Designdateien) ist. Bitte beachten Sie, dass Designs, die mit Origin geliefert wurden, im Verzeichnis \Themes des Origin-Programmverzeichnisses gespeichert werden. Designs, die von Anwendern erstellt wurden, werden im Verzeichnis Themes im Anwenderdateiordner gespeichert.

Außerdem gibt es eine Schaltfläche Schließen.

Schließen

Der Dialog Designs verwalten wird geöffnet.

Bitte beachten Sie, dass Sie doppelt auf den Namen eines anwendererstellten Designs klicken können, um es umzubenennen. Sie können aber nicht den Namen der Systemdesigns ändern.

Die Registerkarte der Diagrammdesigns

Diese Registerkarte enthält eine Tabelle, die alle Systemdiagrammdesigns und benutzerdefinierte Diagrammdesigns aufführt. Mit Hilfe dieser Registerkarte können Sie das ausgewählte Design auf den festgelegten Bereich der Diagramme anwenden.

Das Kontextmenü auf der Registerkarte Diagramm

Die meisten Bearbeitungsvorgänge im Bezug auf Diagrammdesigns befinden sich im Kontextmenü, das geöffnet wird, wenn Sie mit der rechten Maustaste in die Listentabelle der Diagrammdesigns klicken.

Neue hinzufügen

Fügt ein neues leeres Design hinzu.
Bearbeiten

Diese Option ist nur verfügbar, wenn es sich bei dem ausgewählten Design um ein vom Anwender erstelltes handelt. Der Dialog Design bearbeiten wird geöffnet, mit dem Sie das Design modifizieren können.
Löschen

Löscht das ausgewählte Design. Bitte beachten Sie, dass das gelöschte Design aus der Designliste entfernt wird. Sie können jedoch eine Sicherungsdatei von ihm im Ordner Gelöscht im Anwenderdateiordner finden.
Diagramme

Dupliziert ein Design. Bitte beachten Sie, dass, wenn ein Systemdesign dupliziert wird, das neu erstellte Design ein Anwenderdesign sein wird. Systemdesigns können nicht bearbeitet werden. Sie können sie jedoch duplizieren und ihre Kopien bearbeiten.
Kombinieren

Diese Option ist nur verfügbar, wenn mehrere Designs ausgewählt wurden. Sie wird verwendet, um die in den Designs gespeicherten Einstellungen zu kombinieren. Bitte beachten Sie, dass Sie mit den Tasten Strg/Umschalt mehrere Designs auswählen können.

Wenn die ausgewählten Designs verschiedene Werte für die gleiche Eigenschaft haben, Arial beispielsweise für die Schriftart der Spaltenbeschriftung in dem ersten Design ausgewählt ist, während Times New Roman für die Schriftart des zweiten Designs verwendet wird, wird das kombinierte Ergebnis von der Reihenfolge der Designs in der Designliste bestimmt -- nicht von der Reihenfolge, in der Sie diese Designs ausgewählt haben. Das Design oben in der Liste hat Priorität.
Als Systemdesign festlegen

Legt das ausgewählte Design als Systemdesign fest, was bedeutet, das dieses Design auf allen neu erstellten Diagramme angewendet wird. Das Design wird markiert, wenn es als Systemdesign festgelegt wurde.
Systemdesign löschen

Diese Option ist nur verfügbar, wenn ein Design als Systemdesign festgelegt wurde. Sie wird verwendet, um das Systemdesign zu löschen.
Tooltipps zeigen

Aktiviert/deaktiviert die Anzeige des Tooltipps im Hilfsmittel Designs verwalten.

Die Gruppe Design anwenden auf

Diese Gruppe von Bedienelementen wird verwendet, um das Design auf einen festgelegten Bereich von Diagrammen anzuwenden.

Auswahlliste Design anwenden auf Diese Liste legt den Bereich der Diagramme fest, auf den die in dem ausgewählten Design gespeicherten Einstellungen angewendet werden. Die Optionen sind:
  • Aktuelles Diagramm
    Design auf aktives Diagramm anwenden Wenn die aktive Seite keine Diagrammseite ist, wird diese Option verborgen.
  • Diagramme im Ordner
    Wendet das Design auf alle Diagramme des aktuellen Ordners an.
  • Diagramme im Ordner (Rekursiv)
    Das Design wird auf alle Diagramme im aktiven Ordner und in den Unterordnern angewendet.
  • Diagramme im Ordner (Geöffnet)
    Das Design wird auf alle geöffneten Diagramme im aktiven Ordner angewendet.
  • Diagramm im Ordner (einschließlich Eingebetteten)
    Das Design wird auf alle Diagramme im aktiven Ordner, einschließlich der eingebetteten Diagramme, angewendet.
  • Diagramme im Projekt
    Das Design wird auf alle Diagramme des Projekts angewendet.
  • Festgelegte Diagramme
    Definieren Sie manuell, auf welche Diagramme es angewendet werden soll. Wenn diese Option ausgewählt ist, wird unten ein Bearbeitungsfeld angezeigt, in dem Sie die Diagramme festlegen können.
    Klicken Sie auf die Schaltfläche Diagrammbrowser öffnen, um den Diagrammbrowser zu öffnen und die gewünschten Diagrammfenster aufzulisten und auszuwählen.
  • Diagramme im aktuellen Blatt
    Wenden Sie das Design auf die eingebetteten Diagramme im aktuellen Blatt an. Wenn die aktive Seite keine Arbeitsmappe ist, wird diese Option verborgen.
  • Aktiver Layer
    Wenden Sie das Design auf das aktive Layer an. Wenn die aktive Seite keine Diagrammseite ist, wird diese Option verborgen.
  • Festgelegte Layer
    Definieren Sie manuell, auf welche Diagrammlayer es angewendet werden soll. Wenn diese Option ausgewählt ist, wird unten ein Bearbeitungsfeld angezeigt, in dem Sie die Diagrammlayer angeben können.
    Klicken Sie auf die Schaltfläche Layerbrowser öffnen, um den Layerbrowser zu öffnen und die gewünschten Diagrammlayer aufzulisten und auszuwählen.
Jetzt anwenden Wendet das Design sofort auf die Diagramme an.
Rückgängig: Anwenden Macht den vorherigen Vorgang rückgängig.

    Wendet das Design auf alle Blätter im Projekt an.

    Definieren Sie manuell, auf welche Arbeitsblätter es angewendet werden soll. Wenn diese Option ausgewählt ist, wird unten ein Bearbeitungsfeld angezeigt, in dem Sie die Blätter festlegen können.
