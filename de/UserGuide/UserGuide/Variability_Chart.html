Datenanforderungen

Wählen Sie eine Y-Spalte mit mindestens einer Faktorspalte (bis zu 4 Faktoren).

Diagramm erstellen

Um eine Streuungskarte zu erstellen:

  1. Wählen Sie mindestens eine Arbeitsblattspalte (oder einen Bereich aus mindestens einer Spalte).
  2. Wählen Zeichnen > Statistisch : Streuungskarte, um den Dialog plot_vari zu öffnen.

Der Dialog plot_vari

Registerkarte Eingabe

Daten Wählen Sie die zu zeichnenden Quelldatenspalten aus.
Faktoren Wählen Sie die Spalten als Faktoren. Diese Spalten werden als Gruppierungsinformationen betrachtet. Falls mehrere Spalten ausgewählt wurden, wird die Gruppierung verschachtelt.

Registerkarte Eigenschaften

Zellenmittelwert verbinden Legen Sie fest, ob die Mittelwerte des letzten Faktor verbunden werden sollen.
Mittelwertlinien für Faktoren zeigen Legen Sie fest, ob die Mittelwertlinien für Faktoren gezeigt werden.

Die hier aufgeführten Faktoren sind wie die Faktoren organisiert, die Sie auf der Registerkarte Eingabe aktiviert haben. Sie können die gewünschten Faktoren wählen, um die Mittelwertlinien zu zeigen.
Gesamtmittelwertlinie zeigen Legen Sie fest, ob die Gesamtmittelwertlinie im Diagramm gezeigt werden soll. Falls diese Option aktiviert ist, wird eine Referenzlinie bei Mean(plot(*,y)) gezeigt.
Gesamtmedianlinie zeigen Legen Sie fest, ob die Gesamtmedianlinie im Diagramm gezeigt werden soll. Falls diese Option aktiviert ist, wird eine Referenzlinie bei Meandian(plot(*,y)) gezeigt.

Registerkarte SD-Karte

Standardabweichungsdiagramm zeigen Legen Sie fest, ob das Standardabweichungsdiagramm unter der Streuungskarte gezeigt werden soll.
Mittelwert der Standardabweichung zeigen Legen Sie fest, ob der Mittelwert der Standardabweichungswerte des letzten Faktors gezeigt werden soll.
Mittelwertlinien für Faktoren zeigen Wählen Sie den gewünschten Faktor, um die Mittelwertlinien auf dem Standardabweichungsdiagramm zu zeigen. Die Eingabefaktoren werden hier aufgeführt.

Registerkarte Anzeige

Einzelne Datenpunkte zeigen Legen Sie fest, ob die einzelnen Datenpunkte gezeigt werden sollen anstatt nur die Whiskerlinien. Das Aktivieren der Option setzt den Boxtyp auf Box + Datenüberschneidung und den Boxstil auf Keine Box, falls Box zeigen deaktiviert ist.
Box zeigen Legen Sie fest, ob das Boxdiagramm für jede Gruppe gezeigt wird. Das Aktivieren der Option setzt den Boxstil auf Box.
Bereichslinie zeigen Legen Sie fest, ob die Bereichswhiskerlinie für jede Gruppe gezeigt werden soll.
Vorlage Per Standard wird die Vorlage "Variability" (Streuung) automatisch verwendet. Wenn Sie eine andere Vorlage verwenden möchten, deaktivieren Sie Auto und navigieren Sie zu der entsprechenden Vorlage und wählen sie.
Ausgabedaten/Ausgabediagramme Geben Sie die Zeichnungsdaten und Diagramme in unterschiedlichen Blättern aus.

Sobald Sie auf OK geklickt haben, um das Diagramm zu erstellen, können die Zeichnungsdaten und Ergebnisdiagramme in den festgelegten Blättern ausgegeben werden. Per Standard werden zwei neue Blätter mit den Namen "VariabilityPlotData#" und "VariabilityGraph#“ in der aktuellen Arbeitsmappe erstellt, um die Zeichnungsdaten und Ergebnisdiagramme zu speichern.

Vorlage

variability.otpu (installiert im Origin-Programmordner)

Hinweis

Die Streuungskarte kann verwendet werden, um die Beziehungen zwischen Faktoren und einer Antwort zu untersuchen und die Ursachen für die Streuung zu analysieren. Eine Streuungskarte zeichnet die Daten und Mittelwerte für jede Stufe der Gruppierungsfaktoren mit allen Zeichnungen nebeneinander und Feld für Feld.

Origin zeichnet die Streuungskarten für bis zu 4 Faktoren. Sie können das Standardabweichungsdiagramm selektiv in die Streuungskarte zeichnen, wobei die Standardabweichung für jede Faktorstufenkombination angezeigt wird.

Hinweis: Sie können auf die Schaltfläche Verwandte Apps unten im Dialog kicken, um die zugehörige App "Gage Study“ zu suchen und zu installieren.

Beispiel

In diesem Beispiel wählen wir 2 Faktoren im Feld Faktoren des Dialogs plot_vari: "Identification.Classification" und "Identification.Year“ (mit roter Box markiert).

Entstapeln Sie die Quelldatenspalten mit Faktor 2, einschließlich dem Spaltenfaktor 1 (mit pinker Box markiert); mit 4 entstapelten Spalten zeichnen Sie die Boxdiagramme nur mit Bereichswhiskern und verbundenen Mittelwertpunkten. Setzen Sie im Dialog Details Zeichnung den Faktor 1 als die Spalte Horizontales Feld auf der Registerkarte Feld, um die Zeichnung in zwei Felder aufzuteilen.

Verwenden Sie die Gruppierungsspalten, um die Quellantwortspalte "City mpg" in 8 Gruppen aufzuteilen, und die Standardabweichung für jede Gruppe zu berechnen. Zeichnen Sie die Standardabweichungswerte als ein Punkt-Liniendiagramm unter der Streuungskarte. Sie können sehen, dass die Standardabweichungen in diesem Beispiel relativ konsistent zwischen den Werten ("Automatic transmission" und "Manual transmission") des Faktors 1 variieren.
