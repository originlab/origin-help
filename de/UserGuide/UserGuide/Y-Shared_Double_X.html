Datenanforderungen

Wählen Sie mindestens XXY-Spalten aus, um ein Diagramm mit doppeltem X zu erstellen, wobei zwei X-Achsen das gleiche Y teilen.

Diagramm erstellen

Wählen Sie die gewünschten Daten aus.

Wählen Sie Zeichnen > Mehrere Felder, mehrere Achsen: Doppeltes X mit geteiltem Y im Menü.

Hinweis: Wenn Sie so ein Diagramm zum ersten Mal zeichnen, müssen Sie die Online-Vorlage Y-Shared Double-X Plot auf Aufforderung herunterladen.

Vorlage

YSharedDoubleX.otpu (im Origin-Programmordner installiert)

Hinweise

Dieses Diagramm enthält nur einen Layer mit einem Linien-Punktdiagramm mit X1 auf der unteren Achse und X2 auf der oberen Achse.
