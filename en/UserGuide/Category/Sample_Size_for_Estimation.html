17.9.9 Sample Size for Estimation

Sample-Size-Estimation

Use this feature to estimate the sample size by a specific margin of error for confidence intervals for the mean, standard deviation, variance, proportion, rate, or mean (Poisson), or estimate the margin of error based on the sample size.

Generally speaking, Power and Sample Size tools can be used in two ways: sample-size estimation and power computation. The former can be used to determine the size of the sample to ensure a certain power when you are designing an experiment. The latter can be used to estimate the accuracy of the result after you have conducted an experiment on a specific number of samples.


To estimate the sample size by this tool:

  1. Select Statistics: Power and Sample Size: Sample Size for Estimation. This opens the sample_sizes_estimation dialog box.
  2. Select the Parameter and enter the value for the parameter.
  3. From the Calculate list, choose Power or Sample Size, then choose your remaining options.
  4. Upon clicking OK, an analysis report sheet is generated.
