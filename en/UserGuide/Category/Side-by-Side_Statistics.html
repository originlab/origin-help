Introduction

Side-by-Side Statistics is used to statistics on columns with group and output result on the right of each row of data.


The result can:

  • output combination groups of right side of source data.
  • output corresponding statistics on the right of each row of data.

Performing Statistics Side-by-Side

To calculate descriptive statistics Side-by-Side:

  1. Select Statistics: Descriptive Statistics: Side-by-Side Statistics in the menu.
  2. Specify the data range to be analyzed in the Input Data branch and Groups branch. Then specify descriptive statistics in the Quantities branch.
  3. Specify output result to each combination groups on right side of source data, or output corresponding statistics on the right of each row of data
  4. Upon clicking OK, a result will be generated on the right of source data.
