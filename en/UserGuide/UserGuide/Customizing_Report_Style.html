27.7.7 Customizing Report Style

Report-Style

The Report Style dialog box provides interface to control Origin's report sheet table style.

To open this dialog,
  • Select Preference: Report Style: Open Dialog

You can use Preview panel to check the customized setting.

When you click OK to save the new style, it will be set to the default report sheet style. New report sheet will follow the new style. For the excited report sheet style, you can recalculate to update the existed report sheet’s style.


To use system dialog theme

Origin provides multiple system theme of the report style. You can select main menu Preference: Report Style: [Theme Name] directly to change the report sheet style’s theme.
