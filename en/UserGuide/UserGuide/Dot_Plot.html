Data Requirements

Select at least one column to create Wilkinson dot plots or histogram dot plots.

Creating the Graph

To create a Dot Plot:

  1. Highlight at least one worksheet column(or a range from one column).
  2. Select Plot > Statistical : Dot Plot.

Template

statdot.optu (installed to the Origin program folder)

Notes

When you select multiple input columns, each column will be plotted as a dot plot arranging points from the base line and these dot plots will be stacked on Y axis.

Version Information

Minimum Version: Origin 2025
