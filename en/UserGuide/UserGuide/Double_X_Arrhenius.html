Data Requirements

Select temperature and rate constant as XY column to create Arrhenius plot.

Creating the Graph

Select required data.

Select Plot > Multi-Panel, Multi-Axis: Arrhenius from the main menu.

Note：Selecting this menu will add two more columns in the source sheet, one column is 1/(first column), 2nd one is ln(2nd column), take these two column as XY to plot scatter plot in 1st layer, add 2nd layer with x with custom formula.

Template

ARRHENIUS.otpu (installed in the Origin program folder).

Notes

This graph contains two layers, one of which plots a line+scatter plot with newly-calculated columns and another of which shows the top X with custom formula.

Version Information

Minimum Version: Origin 2026
