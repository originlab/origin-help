Data Requirements

Select at least XY columns to create a Double X plot, where the 2nd X axis is functionally mapped to the first.

Ideal for displaying data in different units, such as temperature (°C/°F/K), wavelength and wavenumber.

Creating the Graph

Select required data.

Select Plot > Multi-Panel, Multi-Axis: Double X – Custom Formula from the main menu.

A dialog will pop up to let you set the secondary axis:

There are two method to functionally mapped the double X axes: Independent Ticks and Share Ticks.

  • Independent Ticks: Selecting this type to allow you to enter Direct Formula and Inverse Formula to represents the secondary X axis in another layer without Y axis. The tick position is decided by the secondary axis itself. You can specify whether share Y range between these two layers.
    For more details, you can refer to the scale type Custom Formula in Scale tab of Axis dialog.
  • Share Ticks:Selecting this type to show shared ticks on secondary X axis of current layer, and the tick labels are decided by the formula you entered.

Template

dxcform.otpu (installed in the Origin program folder).

Notes

This graph contains two layers if you select Functionally Mapped - Independent Ticks as Double X Type, the secondary X axis will be added to the second layer. If you selected Functionally Mapped - Share Ticks, only one layer will be plotted with both bottom X axis and Top X axis which share tick position and with different tick labels related to the specified formula.

Version Information
