3.4 Find in Project Files

Find-in-Project

Origin supports a Find in Project Files tool to search content inside Origin project files (.opju, opj) such as sheet name, notes and metadata.

To open the Find in Project Files dialog:

  • Select Edit: Find in Project Files... in the Origin main menu.

or

  • In the upper panel of Project Explorer, right-click and choose Find in Project Files in the context menu.

or

  • Click the Find in Project Files button, in the Project Browser dialog.

or

  • Press the shortcut key Ctrl+Shift+F12 in Origin project file.

or

  • Right-click on the Origin gray area workspace, and then choose Find in Project Files in the context menu.

or

  • Hold Alt key and double click on workspace

or

  • In Recent Origin Files dialog, right-click on the file and select Find in Project Files....

Find

This find enter box supports wildcards (* and ?).

Case Sensitive Button

Use this button to specify whether or not the search is case-sensitive.

Match Whole Word Button

Use this button to specify whether or not the search is matching whole word.

Search In

Specify searching range in All/ File Name/ Books/ Graphs/ Notes/ Folder Names

Exclude Large Files

Select this check-box to exclude the large files when search to make the searching processing fast. And file size limitation can be entered in the right side box.

Look In

Recent Project

Select this check-box to search content in the recent project files.

File Location

You can specify the path of the file location. Then it will search content in the project files in this specified location.

Including Subfolders

Select this check-box to search content not only on the folder of the specified path, but also include its subfolders.

List Box

Detailed View

  • Uncheck the Detailed View option, it will show Path/ Name /Modified / Size/ Count/ Found Info columns in the list box.
  • Check the Detailed View option, it will show Path/ Name /Modified/ Size/ Location/ Element/ Type/ Folder columns in the list box.

Color Rows by File

After check the Detailed View option, you can click this check box to distinguish the project information in each row

Sort Columns

Click on the header of the columns in the list box to sort the searching result by the column.

Pause or Stop

If the finding processing is slow or pass a long time, you can click the Pause or Stop button to stop it. After click Pause button, you can click the Resume button to resume the finding processing.
