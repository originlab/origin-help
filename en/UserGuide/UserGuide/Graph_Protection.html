9.7 Protecting Your Graph

Graph-Protection


Origin supports protecting a single graph to prevent it from being edited or deleted by mistake.

  • Select Tools: Protection: Graph Protection menu(or press Ctrl + F3 hot key) to open the Graph Portection Settings dialog.

Once you checked any check box in this dialog to protect the graph, a lock icon will show before the graph name on its window title bar, in Project Explorer and Object Manager.

Mouse Click

Check this box to prevent the mouse from clicking to select a graph elment to edit.

Graph Deletion

Check this box to prevent from deleting the graph, even when you choose Delete context menu in Project Explorer or click the close button on window title bar.

Scale Change

This option is hidden by default. You can set the system variable @SPS=1 to show it in the dialog. Check this check box will prevent tools or hotkeys that can change the axis scale, such as the Scale in/Out tool, Rescale tool, Zoom-Panning hotkey(for 2D graphs: Z/X key; for 3D graphs: Z/D/X key).
