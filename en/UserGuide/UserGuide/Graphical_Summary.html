Introduction

This tool can be used to create the graphical summary report of the input data with grouping information. The report contains histogram plots, box charts, confidence intervals plots and result tables with statistical information.

Performing Graphical Summary

  • Select the input data columns which you want perfrom basic statistics analysis on.
  • Choose Statistics > Descriptive Statistics: Graphical Summary menu to open the tool.
  • Select grouping column(s) for Group(s) option, enter a value in Confidence Level(%) box and select a desired arrangement under Graph Arrangement list.
  • Click OK to perfrom descriptive statistics and create the graphs.

Dialog Controls

Input Choose the input column(s).
Groups Choose grouping column(s) to divide the input data into multiple groups.
Confidence Interval(%) Specify conf. level of confidence interval
Graph Arrangement Specify how to arrange the statistical graphs.
  • Separate Graphs: Each variable for each group will be plotted into a separate graph. Each graph includes a histogram with distribution curve, a confidence intervals plot and a probability plot, with three result tables "Description Statistics", "##% Confidence Intervals" and "Normality Test". The graph title shows the variable and group information.
  • Per Variable: Each Variable will be plotted into a separate graph. The groups will be plotted into different veritical panels. Each graph includes histograms with distribution curve, box charts with outliers, confidence intervals plots, and a table summarizing the results by row, showing statistics for each group.
  • Per Group: Each group will be plotted into a separate graph. The variables will be plotted into different veritical panels. Each graph includes histograms with distribution curve, box charts with outliers, confidence intervals plots, and a table summarizing the results by row, showing statistics for each variable.
  • All in One:All variables and all groups will be plotted into one graph. Each subset(Group*Variable) will be plotted into different veritical panels. This graph includes histograms with distribution curve, box charts with outliers, confidence intervals plots, and a table summarizing the results by row, showing statistics for each subset.
Output Plot Data Specify where to output the plot data.
Output Graphs Specify where to output the result graphs.
