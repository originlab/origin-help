Data Requirements

Select at least one column as input, prepare at least one group column to create grouped dot plots.

Creating the Graph

To create a Grouped Dot Plot:

  1. Highlight at least one column(or a range from at least one column).
  2. Select Plot > Categorical : Grouped Dot Plots to open the plot_gdot dialog. Select one grouping column at least in the Group box.

Template

statdot.optu (installed to the Origin program folder)

Notes

When you select multiple input columns, each column will be plotted as a grouped dot plot arranging points from the base line and these grouped dot plots will be stacked on Y axis.

More detials, please refer to the Creating Dot Plot page.
