Data Requirements

Select X(θ)Y(r)YY columns to create vector plot (or quiver plot) in polar coordinate. 2nd Y represents angle, 3rd Y represents magnitude.

Creating the Graph

Select Plot > Specialized: Polar Vector θRAM

Template

PolarVectorAM.otpu (All above are installed to the Origin program folder).

Notes

This is a XYAM vector in the polar coordinate, the θRAM columns contain the angle and radial coordinates of start points, the offset angle and magnitude of the vectors.

You can go to the Vector tab of Plot Details to customize the vectors.

Version Information

Minimum Version: Origin 2025b
