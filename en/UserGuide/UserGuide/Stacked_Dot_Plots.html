Data Requirements

Select at least two columns to create stacked dot plots.

Creating the Graph

To create a Dot Plot:

  1. Highlight at least two worksheet columns(or a range from at least two columns).
  2. Select Plot > Statistical : Stacked Dot Plots.

Template

StackedDot.optu (installed to the Origin program folder)

Notes

All input multiple columns will be plot as one dot plot and stacked cumulatively.

More detials, please refer to the Creating Dot Plot page.
