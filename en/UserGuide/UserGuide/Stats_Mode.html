2.8 Stats Mode

Stats-Mode

What is Stats Mode?

Statistics Mode is introduced from Origin 2025b to focus on statistics routine. This mode cleans up Origin interface by hiding most of the graph toolbar buttons and advanced analysis tools, and reorganized Graph menu to make statistical graphs more accessible and easier to navigate. It also pre-installs some popular statistics Apps, e.g. DOE.


Since stats mode keeps only some basic menus and buttons, if you cannot find a tool from the interface, please use search box to find and execute it.

How to enable Stats Mode?

  • Enable Stats Mode during Installation
    When installing Origin, you have options to pre-install a collection of Origin's statistical apps and enable the Stats Mode on the "Select Graphical User Interface (GUI) Mode" page.

    Once done, Origin will be launched in the Stats Mode by default.
    Note: Refer to this page for step-by-step instructions of Origin installation.
  • If you have installed Origin with the '''Default GUI''' option,
    You can always switch to Stats Mode by “modifying Origin”:
    1. Select menu Repair or Uninstall Origin #### from Windows Start menu.
    2. Select Modify on the Welcome page.
    3. Click Next all the way through until "Select Graphical User Interface (GUI) Mode" page. Select Stats GUI radio button.
    4. Click Next as promoted to complete the installation.
    Notes:

    When you check the Stats GUI checkbox, one of the Apps that will be pre-installed is Design of Experiments (DOE). This App requires R Software version 4.4.1 to be installed BEFORE OriginPro is installed. Please use the link below to get this particular version of R software:

    • Index of /bin/windows/base/old/4.4.1

How to turn off Stats Mode?

In any case you can to switch back to the classic mode:

  • Select menu Preferences: Mode: Default
