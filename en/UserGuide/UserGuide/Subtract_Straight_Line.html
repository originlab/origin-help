16.18 Subtract Straight Line

Subtract-Straight-Line

Description

This tool can be used to subtract a straight line from a data set. The straight line is determined by two points. If a graph window is active, you can use this tool from the menu.

To subtract straight line
  • When a graph window is active, select Analysis: Mathematics: Subtract Straight Line

or

  • When a worksheet is active, run subtract_line -d; in the Script Window to open the dialog.

Dialog Options

Recalculate

Controls recalculation of analysis results

  • None
  • Auto
  • Manual

For more information, see: Recalculating Analysis Results

Input

Specifies the XY data range or curve that will be subtracted.

For help with range controls, see: Specifying Your Input Data

Define Line By

Select the method for defining the straight line:

  • Two Point:
    Specifies the X and Y values of the beginning and ending points for the straight line.
    These two points also can be picked in the graph by the Pick Points button. Double click on the graph to pick the points.
  • Fit Line from Selected Range:
    Specifies the line range from a plot in the graph or a column in the worksheet.
Output

Specify the output data range. The output curve will be added in the graph.

For help with the range controls, see: Output Results.
