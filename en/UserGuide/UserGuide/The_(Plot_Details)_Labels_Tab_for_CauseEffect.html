9.3.7.50 The (Plot Details) Labels Tab for Cause-and-Effect

PD-Labels-tab-CauseEffect

This tab is used to do the labels settings for the Cause-and-Effect diagram.

Effect

You can use the controls to set the font, add a frame or wrap the text for the label of effect.

Main Causes

You can use the controls to set the font, add a frame or wrap the text for the labels of main causes.

Sub Causes

You can use the controls to set the font, add a frame or wrap the text for the labels of sub causes.
