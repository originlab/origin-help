Font

Specify the font, font size and other styles for all the labels.

Position

Specify the position of the labels on rectangles.

Format

Specify the label content, display format for the labels.

Overflow

Specify how to hanlde the labels overflowed.

Hide Label if Percentage Less than(%)

Hide the labels of the rectangles, when its percentage (in total weight of all Y values) is less than a specified value.

Group Labels

Show

Specify the group labels of which level(s) will be shown.

Placement

Specify where to place the group labels.

Position

Specify the location of the group labels.

Font

Same as the Font control for the labels on the rectangles(the labels for last level). By default, it will automatically follow the setttings of above Font control.

Background

Set background color for the group labels. You are allowed to select an increment color list to apply different incremental colors to different groups/subgroups.

Frame

When Placement=Overlay, you can add a frame to the group label.

Overflow

Specify how to handle the overflow of the labels. It is same as the Overflow control above.
