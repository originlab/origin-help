Results Log Output

Select the box to output results to Results Log (in addition to that specified in Output Results).

Calculate

Power Compute power for specified sample size.
Sample Size Compute sample size for proposed power.

Test Specification

Null Rate Value of the hypothesized rate.
Alternate Rate Value against which the alternate rate is tested. Support inputting multiple Alternative Rate or select a column in the Worksheet.
Alpha Probability of rejecting the null hypothesis when it is true.
Hypothetical Power(s) Specify space-separated list of hypothetical power(s). Available when Calculate = Sample Size.
Hypothetical Sample Size(s) Specify space-separated list of hypothetical sample size(s). Available when Calculate = Power.
Alternative Hypothesis

Perform a one- or two-sided test:

Not Equal
Indicates that the two-tail Poisson rate test is used.
Less Than
Indicates that the lower-tail Poisson rate test is used.
Greater Than
Indicates that the upper-tail Poisson rate test is used.

Exposure

Enter a value to divide the sample rate of occurrence into a more useful form.

Power Curve

Select to output power curves with assumption. The power curve plots every combination of power v.s. the Alternative Rate. Power curves are helpful for determining the appropriate sample size or power for the test.

Output Results

Determine where to create the output report worksheet. For more information, see Analysis Report Sheets.
