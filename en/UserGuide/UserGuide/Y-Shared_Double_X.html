Data Requirements

Select at least XXY columns to create a Double X plot, where two X axes share same Y.

Creating the Graph

Select required data.

Select Plot > Multi-Panel, Multi-Axis: Y-Shared Double X from the main menu.

Note：When you plot such a graph for the first time, you have to download the online template Y-Shared Double-X Plot as prompted.

Template

YSharedDoubleX.otpu (installed in the Origin program folder).

Notes

This graph contains only one layer, which plot one line+scatter plot with X1 in bottom axis and X2 in top axis.
