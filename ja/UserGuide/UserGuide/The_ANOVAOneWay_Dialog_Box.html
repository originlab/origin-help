OneWayANOVA-Dialog

再計算

分析結果の再計算の設定を変更します。

  • なし
  • 自動
  • 手動

詳細情報は、 分析結果の再計算をご覧下さい。

入力

入力データの形式として、インデックスまたは素データ値を指定します。

インデックス このモードでは、1つの列に因子変数のレベルが含まれ、対応する応答データは別の列に含まれます。

因子

因子列を指定します。因子列がテキスト列の場合、カテゴリー列に設定されます。

データ

応答データの列を指定します。

：結果プロットで因子のテキスト値の代わりに数値を表示する場合は、因子列の列の属性をグループに設定してください。

素データ値 このモードでは、各因子変数のレベルからの応答データが別々の列に格納されます。

因子

名前
因子名を入力します。
レベルの数
デフォルトは2です。
レベルの数を指定します。各レベルには独自のレベルiコントロールがあります。システム変数 @AML を使用して、ANOVAでサポートされる最大レベル数（デフォルトは25）を制御できます。最大レベル数は、コマンドウィンドウで@AMLに新しい値を割り当てることによって変更できます。
レベルi 名
デフォルト = 因子iレベルi。
レベル名を指定します。自動の場合、ロングネームを使用します。

データ

レベルi
各レベルのデータ列を指定します。データ選択コントロールを使用して、各レベルのデータを選択します。もしくは、データの右側にある参照ボタンをクリックして、ワークシートから選択するか、列を選択をクリックして列ブラウザを開き、列を選択できます。

検出力解析

検出力解析の実行を選択します。

実際の検出力 実際の検出力テストを実行するかどうかを指定します。

出力

グラフ配置

グラフを縦列に配置

出力グラフを配置する列の数を指定します。

グラフ中に同じ形式のプロットを配列

1つのグラフウィンドウに同じタイプのすべてのデータプロットを追加できるようにします。
レポート表 レポートワークシート表の出力場所を指定します。

ブック

目的のワークブックを指定します。出力ブックはブック名にあります。
  • <なし>: レポートワークシート表を出力しません（ブック名ボックスは空で読み取り専用です）。
  • <自動>: 初期入力データ選択の場所（入力データ範囲は異なるブックからのものである場合もあります）。入力データがOriginワークブックにない場合、Originは自動的に適合するOriginワークブックを選択します(通常は新しいワークブックを作成)。
  • <ソース>: ソースデータのワークブックです。
  • <新規>: 新しいワークブックです。
  • <既存のもの>:指定した既存のワークブック

シート

出力先ワークシートです（常に<新規>で名前はSheetName）。

結果ログ

結果ログにレポートを出力するかどうかを指定します。

スクリプトウィンドウ

スクリプトウィンドウにレポートを出力するかどうかを指定します。

ノートウィンドウ

このドロップダウンリストを使って目的のノートウィンドウを指定します。
  • <なし>: どのノートウィンドウにも出力しません。
  • <新規>:新しいノートウィンドウに出力します。ここにノートウィンドウの名前を入力します。
プロットデータ プロットのデータの出力先を指定します（プロットするには、下のプロットオプションを選択する必要があります。）。

ブック

プロットデータを出力するワークブックを指定します。
  • <自動>: ブック名を<なし>にしていなければ、レポート表ブックの設定に従います。
  • <ソース>: ソースデータのワークブックです。
  • <レポート>: レポートワークブックです。
  • <新規>: 新しいワークブックです。
  • <既存のもの>:指定した既存のワークブック

シート

出力先ワークシートです（常に<新規>で名前はSheetName）。

ボックスチャート
平均値プロット (SD をエラーバー表示)
平均値プロット (SE をエラーバー表示)
平均比較プロット これは、事後対比較が実行されたときにのみ利用可能です（平均比較ツリーで制御）。

選択したすべてのメソッドについて、平均比較プロットを出力するかどうかを指定します。

残差プロット 残差プロットは、分散分析の仮定が満たされているかどうかを確認するために使用されます。つまり、残差が正規分布しているか、分散が一定かを確認します。ここでは、Originは以下の4種類のプロットを表示します：
  • 行の順序
  • ヒストグラム
  • フィットY
  • 正規確率
