再計算

分析結果の再計算の設定を変更します。

  • なし
  • 自動
  • 手動

詳細情報は、 分析結果の再計算をご覧下さい。

入力データ

統計: 記述統計: クロス集計とカイ二乗 を選択します。Origin は、クロス集計表と分割表の2種類のデータレイアウトに対応しています。クロス集計表レイアウトと比較すると、分割表は、列因子ごとに集計され、アンスタックされたレイアウトです。

クロス集計表

行 列からデータを選択して、行変数を指定します。
列 列からデータを選択して、列変数を指定します。
レイヤ 列からデータを選択して、レイヤ変数を指定します。レイヤ変数を指定すると、行変数と列変数のクロス集計が、レイヤ変数の各水準で作成されます。
度数 データが度数データモードとして集計されている場合、度数列を選択して度数変数を指定します。

分割表

行因子 列からデータを選択して、行変数を指定します。
列ごとの度数 度数変数を指定するために、列ごとの度数を選択します。（少なくとも2列を選択してください。）
レイヤ 列からデータを選択して、レイヤ変数を指定します。レイヤ変数を指定すると、行変数と列変数のクロス集計が、レイヤ変数の各水準で作成されます。

統計

分割表 このブランチでは、分割表で計算して出力する統計量を指定します。
  • 度数：度数、期待度数
  • パーセント：行度数、列度数のパーセント、総度数のパーセント
  • 残差：残差、標準化残差、調整済み残差。

Note:これらの統計についての詳細は、イントロダクションとアルゴリズムのページをご覧ください。

有意水準 以下の仮説検定で使用する有意水準を指定します。デフォルトの値は0.05です。

検定

カイ二乗検定 分割表内の変数間に有意な関係があるかどうかを判定するために、カイ二乗検定を実行するかどうかを指定します。

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

Fisher の正確確率検定 分割表内の変数間に有意な関係があるかどうかを判定するために、Fisher の正確確率検定を実行するかどうかを指定します。

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

関連性の度合い このブランチでは、分割表内の変数間の関係の強さを評価するために計算する統計量を指定します。このブランチは以下のチェックボックスを含みます。
  • ガンマ
  • Kendall の tau-b、tau-c
  • Phi
  • CramerのV
  • 分割係数
  • SomerのD
  • ラムダ
  • 不確定性係数

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

一致性 このブランチでは、対応のあるペアデータの一致度を測定および検定するために実行する検定を指定します。このブランチは以下のチェックボックスを含みます。
  • カッパ
  • Bowkerの検定([2*2]の場合はマクマネー)

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

その他 その他の検定を実行し、その他の統計量を計算するかどうかを指定します。このブランチは以下のチェックボックスを含みます。
  • Cochran-Mantel-Haenszel
    複数の分割表の関係を解析するかどうかを指定します。これには、条件付き独立性の検定、オッズ比の均質性検定、共通オッズ比が含まれます。
  • 相対危険度
    あるグループでイベントが発生するリスクと、比較グループでイベントが発生するリスクの比である相対リスクを計算するかどうかを指定します。
  • オッズ比
    イベントまたは結果が発生するオッズと、イベントが発生しないオッズの比であるオッズ比を計算するかどうかを指定します。

Note: これらの統計についての詳細は、イントロダクション、アルゴリズム、結果の解釈のページをご覧ください。

出力

結果ログの出力 結果ログを表示するかどうかを指定します。
モザイクプロット モザイクプロットを表示するかどうかを指定します。モザイクプロットでは、各四角形の面積が、X 変数の各水準における Y 変数の割合に比例します。
クロス集計レポート 累積の結果の出力範囲を指定します。クロス集計結果を出力するシートを指定します。デフォルト値は、入力データのワークブック内の新しいシートです。
プロットデータ モザイクプロット用データを出力するシートを指定します。クロス集計結果を出力するシートを指定します。デフォルト値は、入力データのワークブック内の新しいシートです。
