平均の t 検定

検定する平均 検定される仮説の平均値(平均が等しいときは0 )
帰無仮説 帰無仮説: Mean1とMean2には、異なるとはいえません。
対立仮説 対立仮説:

平均1 - 平均2 <> 検定する平均

両側の2標本の検定を使用。平均1と平均2に差がある場合。

平均1 - 平均2 > 検定する平均

上側の2標本の検定を使用。平均1 は平均2 より大きい。

平均1 - 平均2 < 検定する平均

下側の2標本の検定を使用。平均1 は平均2 より小さい。
有意水準 有意水準を指定します。
効果量 Cohen's d値を計算して、実際の平均差（Mean1 - Mean2）と指定した検定する平均との差の大きさである効果量を示します。
信頼区間 平均の差についての信頼区間を計算します。

計算の詳細についてはアルゴリズム（Two-Sample T-Test）を参照してください。

%単位での信頼水準 スペース区切りで信頼水準を入力します。このオプションは信頼区間が選択されている場合にのみ利用可能です。
等分散を仮定 等分散を仮定するかどうかを選択します。

2つのサンプルの分散が等しい、またはサンプルサイズが同じ場合は、このチェックボックスをオンにします。2つのサンプルの分散が異なる、またはサンプルサイズが非常に異なる場合は、このチェックボックスを外して、重大なエラーを避けるようにします。

検出力解析

関心のある最小差 検定する平均からの最小差を指定します。ここに値を入力すると、検出力解析のグラフに、この差を検出するために必要なサンプルサイズが異なる検出力ごとに表示されます。
実際の検出力 検定の検出力（帰無仮説が偽のときにそれを棄却する確率）を計算します。
有意水準 仮説の検出力 または 実際の検出力が選択されている場合にのみ利用可能です。検出力計算のための有意水準 (α値) を指定します。
仮説の検出力 サンプルサイズを変化させた場合の検出力を計算します。
仮説のサンプルサイズ 仮説の検出力 が選択されている場合のみ利用可能です。検出力計算のための仮想サンプルサイズを指定します。

プロット

ヒストグラム ヒストグラムを作成します。
